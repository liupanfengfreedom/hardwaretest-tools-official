MP3 کیسے ریکارڈ کریں؟
فی الحال، یہ ٹول WEBM (اصل معیار) اور WAV (نقصان کے بغیر) کو برآمد کرنے کی حمایت کرتا ہے۔ اگر آپ کو MP3 فارمیٹ کی ضرورت ہے، تو ہم تجویز کرتے ہیں کہ پہلے WAV میں ایکسپورٹ کریں اور پھر ہمارا استعمال کریں۔آڈیو کنورٹر ٹولہم مستقبل کے ورژنز میں براہ راست MP3 برآمد کو سپورٹ کرنے کا ارادہ رکھتے ہیں - دیکھتے رہیں!
میں سسٹم آڈیو کیسے ریکارڈ کروں؟
بطور ماخذ "سسٹم آڈیو" کو منتخب کریں اور REC پر کلک کریں۔ براؤزر شیئر ڈائیلاگ میں، کوئی بھی شیئر ٹارگٹ منتخب کریں (جیسے پوری اسکرین) اور یقینی بنائیں کہ نیچے "شیئر سسٹم آڈیو" کو چیک کریں۔ شروع کرنے کی تصدیق کریں۔
میں سسٹم ساؤنڈ کیوں نہیں پکڑ سکتا؟
غالباً آپ پاپ اپ میں "آڈیو شیئر کریں" کے چیک باکس کو چیک کرنا بھول گئے ہیں۔ نیز، macOS کو بلیک ہول انسٹال کرنے کی ضرورت ہے۔ فائر فاکس سسٹم آڈیو کو سپورٹ نہیں کرتا - کروم/ایج استعمال کریں۔
شیئر ڈائیلاگ میں کون سے اختیارات نظر آتے ہیں؟
آپ کو تین ٹیبز نظر آئیں گے: "کروم ٹیب"، "ونڈو"، "پوری اسکرین"۔ کوئی بھی منتخب کریں، لیکن آپ کو نیچے "شیئر سسٹم آڈیو" باکس کو چیک کرنا چاہیے، پھر شیئر پر کلک کریں۔
کون سے ایکسپورٹ فارمیٹس دستیاب ہیں؟
ترتیبات میں آپ خودکار برآمد کو منتخب کر سکتے ہیں: WEBM (اصل) یا WAV (نقصان کے بغیر)۔ آٹو ایکسپورٹ کے بغیر، ریکارڈنگز بعد میں دستی برآمد کے لیے فہرست میں رہتی ہیں۔
شیڈولڈ اسٹاپ کا استعمال کیسے کریں؟
"شیڈولڈ اسٹاپ" میں ایک نمبر پُر کریں اور یونٹ منتخب کریں (سیکنڈ/منٹ/گھنٹہ)۔ جیسے 5 + منٹ = 5 منٹ کے بعد رکیں۔ 0 یا خالی = دستی اسٹاپ۔
