Ketentuan Layanan

Terakhir diperbarui: Maret 2026

1. Penerimaan Persyaratan

Dengan mengakses situs web ini, Anda setuju untuk terikat oleh Ketentuan Layanan ini.

2. Penggunaan Situs Web

Situs web ini menyediakan alat online dan layanan informasi. Anda dapat menggunakan situs web ini hanya untuk tujuan pribadi dan non-komersial.

3. Kegiatan yang Dilarang

  • Pengikisan atau ekstraksi data otomatis
  • Rekayasa terbalik
  • Mencoba mengganggu fungsi situs web
  • Menggunakan situs web untuk tujuan yang melanggar hukum

4. Kekayaan Intelektual

Semua konten, desain, kode, dan materi di situs web ini adalah milik operator situs web dan dilindungi oleh undang-undang hak cipta yang berlaku.

5. Ketersediaan Layanan

Kami berhak memodifikasi, menangguhkan, atau menghentikan bagian mana pun dari situs web kapan pun tanpa pemberitahuan.

6. Batasan Tanggung Jawab

Kami tidak bertanggung jawab atas segala kerusakan langsung, tidak langsung, insidental, atau konsekuensial yang diakibatkan oleh penggunaan situs web ini.

7. Hukum yang Mengatur

Ketentuan ini diatur oleh hukum Amerika Serikat.

8. Kontak

Jika Anda memiliki pertanyaan mengenai Ketentuan ini, silakan hubungi: contact@starryring.com
