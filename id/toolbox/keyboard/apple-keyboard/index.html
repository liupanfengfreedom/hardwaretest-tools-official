Apple Magic Keyboard & Fitur Penguji Keyboard Apple

Visualisasi Keyboard Lengkap · Kompatibel dengan Keyboard Ajaib

Tata letak keyboard Apple ISO yang lengkap. Baik Anda menggunakan Apple Magic Keyboard, Magic Keyboard, atau keyboard bawaan AppleBook, setiap penekanan tombol akan langsung menyorot tombol terkait. Mendukung tombol fungsi, pengubah, dan kunci internasional untuk umpan balik intuitif.

Deteksi Obrolan Tepat

Secara otomatis menandai waktu setiap penekanan tombol. Saat dua kali penekanan terjadi dalam waktu 80 md, entri log disorot dengan warna merah untuk membantu Anda dengan cepat mengidentifikasi potensi masalah pantulan tombol di Magic Keyboard Anda.

Tip: Interval tombol normal biasanya >100 ms. Obrolan cepat yang tidak normal sering kali disebabkan oleh penuaan sakelar.

Uji Pengubah & Kunci Kombo

Uji Kontrol, Opsi, Perintah, Fn, dan pengubah lainnya secara individual. Tekan beberapa tombol secara bersamaan untuk menguji NKRO (maks tombol bersamaan ditampilkan). Sempurna untuk mengevaluasi mekanisme kupu-kupu atau gunting Magic Keyboard.

Analisis Log & Kode Kunci Waktu Nyata

Setiap pers dan rilis dicatat dengan waktu, kunci, kode, dan nilai yang tepat – ideal untuk pengembang atau pengguna tingkat lanjut yang menganalisis detail input atau latensi Bluetooth di Magic Keyboard.

Kompatibel dengan Semua Browser Modern

Dibangun dengan vanilla JavaScript, tidak diperlukan plugin. Berjalan dengan lancar di Chrome, Edge, Safari, Firefox. Bekerja dengan Magic Keyboard yang terhubung dengan Bluetooth juga.

Reset sekali klik, ulangi pengujian

Klik "Reset Tes" untuk menghapus semua penghitungan dan log, sehingga memudahkan untuk menjalankan beberapa putaran pengujian atau membandingkan keyboard yang berbeda (misalnya, Magic Keyboard vs. pihak ketiga).

Pertanyaan yang Sering Diajukan (FAQ)

Bagaimana cara menguji tombol di Apple Magic Keyboard saya?

Tekan tombol apa saja di Magic Keyboard Anda sambil melihat tata letak keyboard di tengah halaman. Kunci yang sesuai akan disorot, panel status akan menampilkan nama kunci, Kode Kunci, dan nilainya, dan area log akan mencatat setiap penekanan dan pelepasan dengan stempel waktu. Jika kunci tidak merespons, ini mungkin mengindikasikan masalah perangkat keras atau masalah koneksi Bluetooth.

Apa itu deteksi obrolan? Bagaimana cara mengetahui jika tombol Magic Keyboard berceloteh?

Deteksi obrolan memonitor interval waktu antara dua penekanan tombol yang sama. Jika intervalnya kurang dari 80 milidetik (biasanya menunjukkan klik dua kali cepat yang tidak normal), entri log disorot dengan warna merah dan diberi label sebagai peringatan obrolan. Ini membantu Anda mengidentifikasi potensi masalah pantulan sakelar, yang umum terjadi pada beberapa mekanisme keyboard.

Mengapa Apple Magic Keyboard saya tidak menunjukkan respons di log ketika saya menekan tombol?

Pastikan area pengujian aktif (langsung tekan tombol apa saja). Jika masih tidak ada respons, mungkin ada masalah izin browser atau masalah perangkat keras. Coba segarkan halaman atau gunakan browser lain (disarankan Chrome, Edge). Periksa juga apakah OS Anda mengenali keyboard, terutama untuk koneksi Bluetooth. Keyboard Ajaib Berkabel juga didukung.

Bagaimana cara menguji tombol pengubah (Perintah, Opsi, dll.)?

Cukup tekan tombol pengubah (misalnya, Command kiri, Option kanan). Tombol yang sesuai pada keyboard layar akan menyala, dan panel status akan menampilkan nama dan kode kunci. Anda dapat menekan beberapa pengubah secara bersamaan untuk menguji pengenalan tombol kombinasi.

Tata letak keyboard Apple manakah yang didukung? Apakah Apple Magic Keyboard kompatibel?

Alat ini menggunakan tata letak keyboard Apple ISO standar, yang mencakup semua tombol utama termasuk tombol fungsi (F1-F12), tombol angka, huruf, tanda baca, dan pengubah baris bawah (Kontrol, Opsi, Perintah) dan tombol panah. Ini sepenuhnya kompatibel dengan Apple Magic Keyboard (termasuk versi keypad numerik), Magic Keyboard, keyboard internal AppleBook, dan keyboard Apple pihak ketiga lainnya.

Apa pentingnya tombol serentak maksimum (NKRO) untuk Magic Keyboard?

Tombol bersamaan maksimum menunjukkan berapa banyak tombol yang dapat ditekan secara bersamaan dan masih dapat dikenali dengan benar. Bagi para gamer atau pengguna dengan shortcut yang rumit, nilai NKRO yang tinggi adalah hal yang penting. Magic Keyboard biasanya mendukung rollover 6 tombol. Alat ini mencatat jumlah tombol yang ditekan saat ini dan menunjukkan riwayat maksimum selama pengujian, memberi Anda wawasan tentang kemampuan konkurensi keyboard Anda yang sebenarnya.
