Kebijakan Privasi

Terakhir diperbarui: Maret 2026

1. Pendahuluan

Situs web ini menghormati privasi Anda dan berkomitmen untuk melindungi informasi Anda. Kebijakan Privasi ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan menjaga informasi ketika Anda mengunjungi situs web kami.

2. Informasi yang Kami Kumpulkan

Kami tidak memerlukan registrasi pengguna. Namun, informasi tertentu dapat dikumpulkan secara otomatis:

  • alamat IP
  • Jenis dan versi peramban
  • Informasi perangkat
  • Halaman yang dikunjungi dan waktu yang dihabiskan
  • Situs web rujukan

3. kue

Situs web ini menggunakan cookie untuk meningkatkan pengalaman pengguna dan menganalisis lalu lintas. Cookie adalah file teks kecil yang disimpan di perangkat Anda.

4. Google Analitik

Kami menggunakan Google Analytics untuk menganalisis penggunaan situs web dan pola lalu lintas. Google dapat mengumpulkan informasi sesuai dengan Kebijakan Privasinya sendiri.

5. Layanan Periklanan

Situs web ini mungkin menampilkan iklan melalui jaringan periklanan pihak ketiga seperti Google AdSense. Layanan ini mungkin menggunakan cookie dan teknologi serupa untuk menampilkan iklan yang dipersonalisasi berdasarkan minat Anda.

6. Bagaimana Kami Menggunakan Informasi

  • Untuk meningkatkan fungsionalitas situs web
  • Untuk menganalisis tren lalu lintas dan penggunaan
  • Untuk menjaga keamanan dan mencegah penyalahgunaan
  • Untuk menayangkan iklan yang relevan

7. Berbagi Data

Kami tidak menjual informasi pribadi. Data dapat diproses oleh penyedia pihak ketiga tepercaya hanya untuk keperluan analisis, periklanan, dan hosting.

8. Retensi Data

Informasi yang dikumpulkan disimpan hanya selama diperlukan untuk analisis, keamanan, dan kepatuhan hukum.

9. Hak Anda

Bergantung pada lokasi Anda, Anda mungkin memiliki hak berdasarkan undang-undang privasi yang berlaku, termasuk hak untuk mengakses, menghapus, atau membatasi pemrosesan informasi pribadi Anda.

10. Privasi Anak

Situs web ini tidak ditujukan untuk anak-anak di bawah usia 13 tahun.

11. Pembaruan

Kami dapat memperbarui Kebijakan Privasi ini dari waktu ke waktu. Penggunaan situs web secara terus-menerus merupakan penerimaan terhadap perubahan apa pun.

12. Kontak

Jika Anda memiliki pertanyaan mengenai Kebijakan Privasi ini, silakan hubungi: contact@starryring.com
