Pertanyaan yang Sering Diajukan

bagaimana cara merekam MP3?
Saat ini, alat ini mendukung ekspor ke WEBM (kualitas asli) dan WAV (lossless). Jika Anda memerlukan format MP3, kami sarankan untuk mengekspor ke WAV terlebih dahulu, lalu menggunakan format kamialat konverter audiountuk mengubahnya. Kami berencana untuk mendukung ekspor MP3 langsung di versi mendatang – pantau terus!
Bagaimana cara merekam audio sistem?
Pilih "Audio sistem" sebagai sumber dan klik REC. Dalam dialog berbagi browser, pilih target berbagi apa pun (misalnya seluruh layar) dan pastikan untuk mencentang "Bagikan audio sistem" di bagian bawah. Konfirmasikan untuk memulai.
Mengapa saya tidak dapat menangkap suara sistem?
Kemungkinan besar Anda lupa mencentang kotak "Bagikan audio" di popup. Selain itu, macOS perlu menginstal BlackHole; Firefox tidak mendukung audio sistem – gunakan Chrome/Edge.
Pilihan apa yang muncul di dialog berbagi?
Anda akan melihat tiga tab: "Tab Chrome", "Jendela", "Keseluruhan layar". Pilih salah satu, tetapi Anda harus mencentang kotak "Bagikan audio sistem" di bagian bawah, lalu klik Bagikan.
Format ekspor apa yang tersedia?
Dalam pengaturan Anda dapat memilih ekspor otomatis: WEBM (asli) atau WAV (lossless). Tanpa ekspor otomatis, rekaman akan tetap ada dalam daftar untuk diekspor secara manual nanti.
Bagaimana cara menggunakan perhentian terjadwal?
Isi nomor di "Perhentian terjadwal" dan pilih satuan (detik/menit/jam). Misalnya. 5+ menit = berhenti setelah 5 menit. 0 atau kosong = berhenti manual.
