Spesifikasi Teknis

Format dan teknologi yang digunakan:

  • SVG: grafik vektor, dapat diskalakan tanpa kehilangan kualitas
  • PNG: lossless, transparan, ideal untuk grafik web
  • JPEG: kompresi lossy, ukuran file kecil, foto
  • WebP: format modern, kompresi 25‑35% lebih baik dibandingkan JPEG
  • BMP: tidak terkompresi, kualitas maksimal
  • TIFF: standar industri percetakan, mendukung lapisan
  • HEIC/HEIF: Format efisiensi tinggi Apple, 50% lebih kecil dari JPEG
  • Mengubah ukuran & konversi: API Kanvas HTML5 + Pekerja Web; SVG diurai dengan DOMParser; HEIC dengan heic2any; TIFF dengan UTIF.js

Catatan:Semua pemrosesan menggunakan Web Worker untuk menjaga antarmuka tetap responsif. Untuk gambar yang sangat besar, pemrosesan mungkin memerlukan waktu beberapa detik tetapi tidak akan membekukan halaman.

Pertanyaan yang Sering Diajukan (FAQ)

Mengapa menggunakan pengubah ukuran gambar lokal?

Alat online tradisional memerlukan pengunggahan gambar ke server, sehingga membahayakan privasi. Alat kami sepenuhnya berjalan di browser Anda — gambar Anda tetap ada di perangkat Anda. Sempurna untuk gambar pribadi, rahasia, atau sensitif.

Dapatkah saya mengubah ukuran gambar? Bagaimana?

Ya! Pilih mode pengubahan ukuran (persen, lebar tetap, tinggi tetap, atau bentangan bebas), masukkan nilai yang diinginkan, dan klik "Proses". Alat ini akan menghasilkan gambar yang diubah ukurannya secara lokal.

Apa itu HEIC dan mengapa mengubahnya?

HEIC adalah format foto default di iPhone/iPad. Ini menawarkan kompresi yang lebih baik daripada JPEG tetapi tidak didukung secara universal. Alat kami mengonversi HEIC ke JPG/PNG sehingga Anda dapat melihat dan menggunakannya di mana saja.

Apakah ini mendukung konversi SVG?

Tentu saja. Unggah file SVG dan pilih format raster keluaran (PNG, JPEG, WebP, BMP, TIFF). Anda juga dapat mengatur dimensi keluaran untuk mendapatkan ukuran yang Anda butuhkan.

Apakah gratis?

Ya, sepenuhnya gratis. Tanpa registrasi, tanpa biaya tersembunyi. Cukup ubah ukuran dan konversi gambar Anda secara pribadi.

Browser apa yang didukung?

Semua browser modern: Chrome 65+, Firefox 61+, Safari 11.1+, Edge 79+. Browser seluler (iOS Safari, Android Chrome) juga berfungsi.
