Pertanyaan yang Sering Diajukan (FAQ)

Apa itu WebSocket?

WebSocket adalah protokol yang menyediakan komunikasi dupleks penuh melalui satu koneksi TCP, memungkinkan server mengirimkan data ke klien. Alat ini membantu menguji apakah layanan WebSocket Anda berfungsi dengan benar.

Bagaimana cara menguji koneksi WebSocket?

Masukkan URL server WebSocket (misalnya wss://echo.websocket.org), klik 'Hubungkan'. Setelah terhubung, ketik pesan di area input dan klik 'Kirim'. Pesan yang diterima muncul di log.

Mengapa koneksi gagal?

Kemungkinan alasan: alamat server salah, server tidak mendukung WSS (coba WS), pemblokiran firewall, server down, atau pembatasan CORS. Periksa URL dan pastikan server mengizinkan koneksi WebSocket.

Bagaimana cara menambahkan header khusus?

Di bawah 'Pengaturan Koneksi', klik '+ Tambah' untuk memasukkan nama dan nilai header. Header ini dikirim selama jabat tangan WebSocket, berguna untuk autentikasi atau parameter khusus.

Format pesan apa yang didukung?

Teks, JSON, dan Hex. Pilih format, masukkan konten Anda, dan itu akan dikonversi secara otomatis. Mode JSON menyertakan tombol format dan perkecil untuk memudahkan pengeditan.

Dapatkah saya menyimpan lognya?

Ya. Klik tombol 'Ekspor' di area log untuk mengunduh log saat ini sebagai file teks.

Bagaimana latensi dihitung?

Setelah koneksi, alat mengirimkan pesan ping bawaan setiap 10 detik dan mengukur waktu hingga pong diterima, menampilkan latensi saat ini.
