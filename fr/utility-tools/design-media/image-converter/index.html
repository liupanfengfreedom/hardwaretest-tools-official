Spécifications techniques

Formats et technologies utilisées :

  • SVG : graphiques vectoriels, évolutifs sans perte de qualité
  • PNG : sans perte, transparence, idéal pour les graphiques Web
  • JPEG : compression avec perte, petite taille de fichier, photos
  • WebP : format moderne, compression 25 à 35 % supérieure à celle du JPEG
  • BMP : non compressé, qualité maximale
  • TIFF : norme de l'industrie de l'impression, prend en charge les calques
  • HEIC/HEIF : format haute efficacité d'Apple, 50 % plus petit que JPEG
  • Redimensionnement et conversion : API HTML5 Canvas + Web Workers ; SVG analysé avec DOMParser ; HEIC avec heic2any ; TIFF avec UTIF.js

Remarque :Tous les traitements utilisent Web Workers pour que l'interface reste réactive. Pour les images très volumineuses, le traitement peut prendre quelques secondes mais ne figera pas la page.

Foire aux questions (FAQ)

Pourquoi utiliser un redimensionneur d'image local ?

Les outils en ligne traditionnels nécessitent de télécharger des images sur un serveur, ce qui met en danger la confidentialité. Notre outil fonctionne entièrement dans votre navigateur — vos images restent sur votre appareil. Parfait pour les images personnelles, confidentielles ou sensibles.

Puis-je redimensionner les images ? Comment?

Oui ! Sélectionnez un mode de redimensionnement (pourcentage, largeur fixe, hauteur fixe ou étirement libre), entrez les valeurs souhaitées et cliquez sur « Traiter ». L'outil générera des images redimensionnées localement.

Qu'est-ce que HEIC et pourquoi le convertir ?

HEIC est le format photo par défaut sur les iPhones/iPads. Il offre une meilleure compression que JPEG mais n'est pas universellement pris en charge. Notre outil convertit HEIC en JPG/PNG afin que vous puissiez les visualiser et les utiliser n'importe où.

Est-il compatible avec la conversion SVG ?

Absolument. Téléchargez des fichiers SVG et choisissez un format raster de sortie (PNG, JPEG, WebP, BMP, TIFF). Vous pouvez également définir les dimensions de sortie pour obtenir exactement la taille dont vous avez besoin.

Est-ce gratuit ?

Oui, entièrement gratuit. Pas d'inscription, pas de frais cachés. Redimensionnez et convertissez simplement vos images en privé.

Quels navigateurs sont pris en charge ?

Tous les navigateurs modernes : Chrome 65+, Firefox 61+, Safari 11.1+, Edge 79+. Les navigateurs mobiles (iOS Safari, Android Chrome) fonctionnent également.
