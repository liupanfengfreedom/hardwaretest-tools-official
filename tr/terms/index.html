1. Şartların Kabulü

Bu web sitesine erişerek bu Hizmet Şartları'na bağlı kalmayı kabul etmiş olursunuz.

2. Web Sitesinin Kullanımı

Bu web sitesi çevrimiçi araçlar ve bilgi hizmetleri sunmaktadır. Web sitesini yalnızca kişisel ve ticari olmayan amaçlarla kullanabilirsiniz.

3. Yasaklanan Faaliyetler

  • Otomatik kazıma veya veri çıkarma
  • Tersine mühendislik
  • Web sitesinin işlevselliğini bozmaya çalışmak
  • Web sitesini yasa dışı amaçlarla kullanmak

4. Fikri Mülkiyet

Bu web sitesindeki tüm içerik, tasarım, kod ve materyaller web sitesi operatörüne aittir ve geçerli telif hakkı yasalarıyla korunmaktadır.

5. Hizmet Kullanılabilirliği

Web sitesinin herhangi bir bölümünü önceden bildirimde bulunmaksızın herhangi bir zamanda değiştirme, askıya alma veya durdurma hakkını saklı tutuyoruz.

6. Sorumluluğun Sınırlandırılması

Bu web sitesinin kullanımından kaynaklanan doğrudan, dolaylı, tesadüfi veya sonuç olarak ortaya çıkan zararlardan sorumlu olmayacağız.

7. Geçerli Kanun
