Frequently Asked Questions (FAQ)

What is password entropy and why does it matter?

Password entropy measures randomness and unpredictability, expressed in bits. Higher entropy means stronger passwords that are harder to crack. Formula: entropy = log₂(charset_size ^ password_length). For example, a 12-character password from a 94-character set yields ~78.5 bits of entropy.

Why would I exclude certain characters?

Excluding characters improves usability: 1) Avoid confusion (0/O, 1/l, 5/S). 2) Some systems restrict special symbols. 3) Easier input on mobile devices. However, note that excluding characters reduces charset size and may slightly impact security.

Are generated passwords truly secure?

Yes. CyberPass Pro uses the browser's Crypto.getRandomValues() API — a cryptographically secure random number generator. All operations happen locally; passwords never leave your device. With a sufficiently large character set, generated passwords are highly secure.

How should I save generated passwords?

Always use a reputable password manager (Bitwarden, 1Password, LastPass, etc.) to store passwords securely. Never save passwords in plain text files or share them unsafely. This tool provides copy & export features for easy integration with password managers.

Is this tool free?

Yes! CyberPass Pro is completely free to use, with no registration or login required. Our goal is to provide high-quality security tools to help everyone improve online account safety.

Can I export generated passwords?

Absolutely. You can export the current password or batch results. Export formats include plain text and CSV-like format for easy import into password managers or other applications.
