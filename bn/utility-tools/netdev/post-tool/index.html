🚀 অনলাইন পোস্ট টুল / API টেস্টার

বিশুদ্ধ ফ্রন্টএন্ড HTTP অনুরোধ ডিবাগিং টুল, একাধিক HTTP পদ্ধতি এবং ডেটা বিন্যাস সমর্থন করে

অনুরোধ কনফিগারেশন

প্রতিক্রিয়া ফলাফল























Tool বৈশিষ্ট্য






বিশুদ্ধ ফ্রন্টেন্ড বাস্তবায়ন


সম্পূর্ণরূপে ব্রাউজারে চলে, কোনো সার্ভারের প্রয়োজন নেই। সমস্ত HTTP অনুরোধ ব্রাউজার ফেচ API এর মাধ্যমে পাঠানো হয়, ডেটা গোপনীয়তা এবং নিরাপত্তা নিশ্চিত করে।






একাধিক HTTP পদ্ধতি


POST, GET, PUT, DELETE, PATCH পাঁচটি HTTP পদ্ধতি সমর্থন করে, মৌলিক RESTful API বিকাশের চাহিদা পূরণ করে।






মাল্টি-ফরম্যাট ডেটা সমর্থন


JSON এবং ফর্ম ডেটা ফরম্যাট সমর্থন করে, বিভিন্ন ধরনের API ইন্টারফেস পরীক্ষা করার জন্য সুবিধাজনক। JSON সম্পাদক সিনট্যাক্স হাইলাইটিং এবং বৈধতা প্রদান করে।






কাস্টম অনুরোধ শিরোনাম


নমনীয়ভাবে অনুরোধ শিরোনাম কনফিগার করে, সাধারণ শিরোনাম সমর্থন করে যেমন বিষয়বস্তু-প্রকার, অনুমোদন, ব্যবহারকারী-এজেন্ট, জটিল API পরীক্ষার প্রয়োজনীয়তা পূরণ করে।






URL ইতিহাস রেকর্ড


স্বয়ংক্রিয়ভাবে সম্প্রতি ব্যবহৃত URLগুলি সংরক্ষণ করে, দ্রুত স্যুইচিং এবং বারবার পরীক্ষার জন্য সুবিধাজনক, ডিবাগিং দক্ষতা উন্নত করে৷






রিয়েল-টাইম প্রতিক্রিয়া পূর্বরূপ


তাৎক্ষণিকভাবে HTTP স্থিতি কোড, প্রতিক্রিয়া শিরোনাম এবং প্রতিক্রিয়া বডি প্রদর্শন করে, JSON ফর্ম্যাটেড প্রদর্শন সমর্থন করে, API রিটার্ন ফলাফল বিশ্লেষণ করা সহজ।








প্রায়শই জিজ্ঞাসিত প্রশ্ন




এই টুলটি কি HTTPS সমর্থন করে?


হ্যাঁ, সম্পূর্ণরূপে HTTPS প্রোটোকল সমর্থন করে। আপনি পরীক্ষার জন্য কোনো বৈধ HTTPS URL লিখতে পারেন। টুলটি ব্রাউজার নিরাপত্তা নীতি অনুসরণ করে।






কিভাবে ক্রস-অরিজিন অনুরোধগুলি পরিচালনা করবেন?


ক্রস-অরিজিন অনুরোধের জন্য লক্ষ্য সার্ভারের প্রয়োজন CORS (ক্রস-অরিজিন রিসোর্স শেয়ারিং) হেডার কনফিগার করতে। আপনি যদি ক্রস-অরিজিন ত্রুটির সম্মুখীন হন, নিশ্চিত করুন যে API সার্ভার সঠিক CORS নীতিগুলি কনফিগার করেছে, যেমন অ্যাক্সেস-কন্ট্রোল-অনুমতি-অরিজিন ইত্যাদি।






এটি কি ফাইল আপলোড সমর্থন করে?


বর্তমান সংস্করণ প্রধানত JSON এবং ফর্ম ডেটা বিন্যাস সমর্থন করে। মাল্টিপার্ট/ফর্ম-ডেটা ফরম্যাট (ফাইল আপলোডের জন্য) ভবিষ্যতের সংস্করণে সমর্থিত হবে। আপনি ফাইল আপলোড পরীক্ষার জন্য এই টুলের সাথে FormData API ব্যবহার করতে পারেন।






ডেটা কি সার্ভারে সংরক্ষিত হয়?


না। সমস্ত ডেটা আপনার ব্রাউজারের স্থানীয় স্টোরেজে (স্থানীয় স্টোরেজ) সংরক্ষণ করা হয়, কোনো সার্ভারে আপলোড করা হয় না, আপনার API কী এবং সংবেদনশীল ডেটা নিরাপত্তা নিশ্চিত করে।






এটি কি WebSocket পরীক্ষা সমর্থন করে?


বর্তমান সংস্করণ HTTP/HTTPS অনুরোধ পরীক্ষার উপর ফোকাস করে। WebSocket টেস্টিং টুল ডেভেলপ করা হচ্ছে এবং ভবিষ্যতের সংস্করণে প্রদান করা হবে।






এটি কি HTTP প্রোটোকল শেখার জন্য উপযুক্ত?


শেখার জন্য পারফেক্ট! এই টুলটি স্বজ্ঞাতভাবে HTTP অনুরোধ এবং প্রতিক্রিয়াগুলির সমস্ত উপাদান প্রদর্শন করে, HTTP প্রোটোকল, RESTful API ডিজাইন এবং ডিবাগিং কৌশল শেখার জন্য একটি আদর্শ টুল।
