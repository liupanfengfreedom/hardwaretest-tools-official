Recursos da ferramenta






Implementação de front-end puro


Executa completamente no navegador, sem necessidade de servidor. Todas as solicitações HTTP são enviadas através da API Fetch do navegador, garantindo privacidade e segurança dos dados.






Vários métodos HTTP


Suporta POST, GET, PUT, DELETE, PATCH cinco métodos HTTP, atendendo às necessidades básicas de desenvolvimento de API RESTful.






Suporte de dados multiformato


Suporta JSON e formatos de dados de formulário, conveniente para testar diferentes tipos de interfaces API. O editor JSON fornece destaque e validação de sintaxe.






Cabeçalhos de solicitação personalizados


Configure cabeçalhos de solicitação com flexibilidade, suporta cabeçalhos comuns como Content-Type, Authorization, User-Agent, atendendo a requisitos complexos de teste de API.

Registro de histórico de URL


Salva automaticamente URLs usados ​​recentemente, conveniente para troca rápida e testes repetidos, melhorando a eficiência da depuração.






Visualização da resposta em tempo real


Exibe instantaneamente o código de status HTTP, cabeçalhos de resposta e corpo de resposta, suporta exibição formatada em JSON e fácil de analisar os resultados de retorno da API.








Perguntas frequentes




Esta ferramenta suporta HTTPS?


Sim, suporta totalmente o protocolo HTTPS. Você pode inserir qualquer URL HTTPS válido para teste. A ferramenta segue as políticas de segurança do navegador.






Como lidar com solicitações de origem cruzada?


As solicitações de origem cruzada exigem que o servidor de destino configure cabeçalhos CORS (compartilhamento de recursos de origem cruzada). Se você encontrar erros de origem cruzada, certifique-se de que o servidor API tenha configurado políticas CORS corretas, como Access-Control-Allow-Origin etc.






Suporta upload de arquivos?


A versão atual suporta principalmente JSON e formatos de dados de formulário. O formato multipart/form-data (para uploads de arquivos) será suportado em versões futuras. Você pode usar a API FormData com esta ferramenta para teste de upload de arquivos.






Os dados são salvos no servidor?


Não. Todos os dados são salvos no armazenamento local do seu navegador (localStorage), não sendo carregados em nenhum servidor, garantindo suas chaves API e segurança de dados confidenciais.






Suporta testes WebSocket?


A versão atual concentra-se em testes de solicitação HTTP/HTTPS. A ferramenta de teste WebSocket está em desenvolvimento e será fornecida em versões futuras.






É adequado para aprender o protocolo HTTP?


Perfeito para aprender! Esta ferramenta exibe intuitivamente todos os componentes de solicitações e respostas HTTP, uma ferramenta ideal para aprender o protocolo HTTP, design de API RESTful e técnicas de depuração.
