Especificações Técnicas

Formatos e tecnologias utilizadas:

  • SVG: gráficos vetoriais, escaláveis sem perda de qualidade
  • PNG: sem perdas, transparência, ideal para gráficos da web
  • JPEG: compactação com perdas, tamanho de arquivo pequeno, fotos
  • WebP: formato moderno, compactação 25-35% melhor que JPEG
  • BMP: não compactado, qualidade máxima
  • TIFF: padrão da indústria de impressão, suporta camadas
  • HEIC/HEIF: formato de alta eficiência da Apple, 50% menor que JPEG
  • Redimensionamento e conversão: HTML5 Canvas API + Web Workers; SVG analisado com DOMParser; HEIC com heic2any; TIFF com UTIF.js

Nota:Todo o processamento usa Web Workers para manter a interface responsiva. Para imagens muito grandes, o processamento pode demorar alguns segundos, mas não congela a página.

Perguntas frequentes (FAQ)

Por que usar um redimensionador de imagem local?

As ferramentas online tradicionais exigem o upload de imagens para um servidor, arriscando a privacidade. Nossa ferramenta funciona inteiramente no seu navegador – suas imagens permanecem no seu dispositivo. Perfeito para imagens pessoais, confidenciais ou sensíveis.

Posso redimensionar imagens? Como?

Sim! Selecione um modo de redimensionamento (porcentagem, largura fixa, altura fixa ou estiramento livre), insira os valores desejados e clique em “Processar”. A ferramenta irá gerar imagens redimensionadas localmente.

O que é HEIC e por que convertê-lo?

HEIC é o formato de foto padrão em iPhones/iPads. Oferece melhor compactação que JPEG, mas não é compatível universalmente. Nossa ferramenta converte HEIC em JPG/PNG para que você possa visualizá-los e usá-los em qualquer lugar.

Suporta conversão SVG?

Com certeza. Carregue arquivos SVG e escolha um formato raster de saída (PNG, JPEG, WebP, BMP, TIFF). Você também pode definir as dimensões de saída para obter exatamente o tamanho necessário.

É grátis?

Sim, totalmente gratuito. Sem registro, sem taxas ocultas. Basta redimensionar e converter suas imagens de forma privada.

Quais navegadores são suportados?

Todos os navegadores modernos: Chrome 65+, Firefox 61+, Safari 11.1+, Edge 79+. Os navegadores móveis (iOS Safari, Android Chrome) também funcionam.
