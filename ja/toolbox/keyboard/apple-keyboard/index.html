Apple Magic Keyboard と Apple Keyboard Tester の機能

フルキーボードビジュアル化・Magic Keyboard対応

完全な Apple ISO キーボード レイアウト。 Apple Magic Keyboard、Magic Keyboard、AppleBook 内蔵キーボードのいずれを使用している場合でも、キーを押すたびに対応するキーが即座にハイライト表示されます。ファンクション キー、修飾キー、国際キーをサポートし、直感的なフィードバックを実現します。

正確なびびり検出

キーを押すたびに自動的にタイムスタンプが付けられます。 80 ミリ秒以内に 2 回押すと、ログ エントリが赤で強調表示され、Magic Keyboard の潜在的なスイッチ バウンスの問題をすばやく特定できるようになります。

ヒント: 通常のキー間隔は通常 >100ms です。異常な速いチャタリングは、スイッチの老朽化によって発生することがよくあります。

修飾子とコンボ キーのテスト

Control、Option、Command、Fn、およびその他の修飾子を個別にテストします。複数のキーを同時に押して、NKRO をテストします (最大同時キーが表示されます)。 Magic Keyboard のバタフライまたはシザー機構の評価に最適です。

リアルタイムログとキーコード分析

すべてのプレスとリリースは、正確な時間、キー、コード、およびその値とともに記録されます。これは、Magic Keyboard で入力の詳細や Bluetooth 遅延を分析する開発者や上級ユーザーに最適です。

すべての最新ブラウザと互換性あり

バニラ JavaScript で構築されており、プラグインは必要ありません。 Chrome、Edge、Safari、Firefox でスムーズに動作します。 Bluetooth接続のMagic Keyboardでも動作します。

ワンクリックでリセット、テストを繰り返す

[テストをリセット] をクリックすると、すべてのカウントとログがクリアされ、複数のテスト ラウンドを実行したり、異なるキーボード (例: Magic Keyboard とサードパーティ) を比較したりすることが簡単になります。

よくある質問 (FAQ)

Apple Magic Keyboard のキーをテストするにはどうすればよいですか?

ページ中央のキーボード レイアウトを見ながら、Magic Keyboard の任意のキーを押します。対応するキーが強調表示され、ステータス パネルにはキー名、KeyCode、およびその値が表示され、ログ領域には各プレスとリリースがタイムスタンプとともに記録されます。キーが応答しない場合は、ハードウェアの問題または Bluetooth 接続の問題を示している可能性があります。

チャタリング検出とは何ですか? Magic Keyboard のキーがチャタリングしているかどうかを確認するにはどうすればよいですか?

チャタリング検出は、同じキーを 2 回押す間の時間間隔を監視します。間隔が 80 ミリ秒未満の場合 (通常、異常な素早いダブルクリックを示します)、ログ エントリは赤色で強調表示され、チャタリング アラートとしてラベル付けされます。これは、一部のキーボード メカニズムで一般的な、潜在的なスイッチ バウンスの問題を特定するのに役立ちます。

Apple Magic Keyboard でキーを押しても応答がログに記録されないのはなぜですか?

テスト領域がアクティブであることを確認してください (いずれかのキーを直接押してください)。それでも応答がない場合は、ブラウザの許可の問題またはハードウェアの問題である可能性があります。ページを更新するか、別のブラウザを使用してみてください (Chrome、Edge を推奨)。特に Bluetooth 接続の場合は、OS がキーボードを認識しているかどうかも確認してください。有線Magic Keyboardもサポートされています。

修飾キー (コマンド、オプションなど) をテストするにはどうすればよいですか?

修飾キーを押すだけです (例: 左の Command、右の Option)。オンスクリーン キーボードの対応するキーが点灯し、ステータス パネルにキーの名前とコードが表示されます。複数の修飾キーを同時に押して、組み合わせキーの認識をテストできます。

どの Apple キーボード レイアウトがサポートされていますか? Apple Magic Keyboardは互換性がありますか?

このツールは、標準 ISO Apple キーボード レイアウトを使用し、ファンクション キー (F1 ～ F12)、数字キー、文字、句読点、最下行の修飾子 (Control、Option、Command) および矢印キーを含むすべての主要キーをカバーします。 Apple Magic Keyboard (テンキー バージョンを含む)、Magic Keyboard、AppleBook 内蔵キーボード、およびその他のサードパーティ製 Apple キーボードと完全な互換性があります。

Magic Keyboard の最大同時キー (NKRO) の重要性は何ですか?

最大同時キーは、同時に押しても正しく認識されるキーの数を示します。ゲーマーや複雑なショートカットを持つユーザーにとって、高い NKRO 値は重要です。 Magic Keyboard は通常、6 キー ロールオーバーをサポートしています。このツールは、現在押されているキーの数を記録し、テスト中の過去の最大値を表示することで、キーボードの実際の同時実行能力についての洞察を得ることができます。
