Privacy Image Studio の機能: サイズ変更と変換

📖 複数の形式とインテリジェントな寸法調整をサポートする、プロフェッショナルなローカル画像リサイザーおよびコンバーター。 画像リサイザ 、 画像コンバータ 、 HEIC コンバータ 、 SVG コンバータ 、および ローカル画像処理 を 1 つのツールに統合します。

100% プライバシー保護

すべての画像のサイズ変更と変換はブラウザ内でローカルに行われます。画像がデバイスから流出することはなく、完全なプライバシーが確保されます。

アップロードなし – 機密性の高い画像やプライベートな画像に最適

フルフォーマットのサポート

PNG、JPEG、WebP、BMP、TIFF、HEIC/HEIF、SVG をサポートします。 iPhone HEIC写真またはSVGベクターグラフィックスを変換し、同時にサイズ変更します。

HEIC サポート: あらゆるデバイスで iPhone の写真を簡単に処理

SVG ベクトルのサポート

カスタム出力サイズを使用して、SVG をラスター形式 (PNG、JPEG など) に変換します。あらゆるサイズで高品質のアイコンやロゴを生成するのに最適です。

ベクトルからラスターへ: SVG を PNG、JPEG、WebP などに変換します。

スマート サイズ変更

複数のサイズ変更モード: パーセントによる拡大縮小、幅/高さの固定 (比率を維持)、自由な伸縮。画像の寸法、アスペクト比、解像度を簡単に調整できます。

幅または高さを指定するときにアスペクト比を維持する

バッチ処理

複数の画像を一度にアップロードします。サイズを変更してバッチで変換し、すべてを ZIP アーカイブとしてダウンロードします。

技術仕様

使用されるフォーマットとテクノロジー:

  • SVG: ベクター グラフィックス、品質を損なうことなくスケーラブル
  • PNG: ロスレス、透明度、Web グラフィックに最適
  • JPEG: 非可逆圧縮、ファイル サイズが小さい、写真
  • WebP: 最新の形式、JPEG よりも 25 ～ 35% 優れた圧縮率
  • BMP: 非圧縮、最高品質
  • TIFF: 印刷業界標準、レイヤーをサポート
  • HEIC/HEIF: Apple の高効率フォーマット、JPEG
    • より 50% 小さい
  • サイズ変更と変換: HTML5 Canvas API + Web Workers。 SVG は DOMParser で解析されます。 HEIC と heic2any; UTIF.js を使用した TIFF

注: すべての処理は、インターフェイスの応答性を維持するために Web ワーカーを使用します。非常に大きな画像の場合、処理に数秒かかる場合がありますが、ページがフリーズすることはありません。

よくある質問 (FAQ)

ローカル画像リサイザーを使用する理由

従来のオンライン ツールでは画像をサーバーにアップロードする必要があり、プライバシーが危険にさらされます。私たちのツールは完全にブラウザ内で実行され、画像はデバイス上に残ります。個人的な画像、機密画像、または機密性の高い画像に最適です。

画像のサイズを変更できますか?どうやって？

はい！サイズ変更モード (パーセント、固定幅、固定高さ、または自由伸縮) を選択し、希望の値を入力して、「処理」をクリックします。このツールは、サイズ変更された画像をローカルで生成します。

HEIC とは何ですか?なぜ変換するのですか?

HEIC は、iPhone/iPad のデフォルトの写真形式です。 JPEG よりも優れた圧縮率を提供しますが、広くサポートされているわけではありません。私たちのツールは HEIC を JPG/PNG に変換するので、どこでも表示して使用できます。

SVG 変換をサポートしていますか?

そうですね。 SVG ファイルをアップロードし、出力ラスター形式 (PNG、JPEG、WebP、BMP、TIFF) を選択します。出力寸法を設定して、必要なサイズを正確に取得することもできます。

無料ですか？

はい、完全に無料です。登録や隠れた手数料はありません。画像のサイズ変更と変換をプライベートで行うだけです。

サポートされているブラウザは何ですか?

すべての最新ブラウザ: Chrome 65 以降、Firefox 61 以降、Safari 11.1 以降、Edge 79 以降。モバイルブラウザ (iOS Safari、Android Chrome) も動作します。
