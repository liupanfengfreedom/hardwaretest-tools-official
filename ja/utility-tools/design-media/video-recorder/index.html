オンライン画面録画ツールの使用手順

このオンライン画面録画ツールは、Web ページのデモンストレーション、製品の操作、コースの説明、会議クリップ、問題再発ビデオの録画に適しています。録画プロセスはブラウザの画面キャプチャ機能に基づいて完了し、ビデオ クリップは現在のデバイス上でローカルに生成され、直接プレビュー、ダウンロード、または削除できます。

録画を開始する前に、画質、フレーム レート、エクスポート形式を選択できます。音について説明する必要がある場合は、まずマイクをオンにしてください。本物の人物をカメラに映す必要がある場合は、最初にカメラの電源をオンにします。そうすれば、ツールは録画されたビデオの右下隅にカメラの画像を重ね合わせます。

ブラウザ共有ポップアップウィンドウの詳細説明

「録画開始」をクリックすると、Chrome または Edge により画面共有選択ウィンドウがポップアップ表示されます。画面全体、アプリ ウィンドウ、またはブラウザー タブを録画することを選択できます。システムサウンドを録音するには、ポップアップウィンドウの下部にある「オーディオを共有」または「システムオーディオも共有」にチェックを入れることが重要です。

よくある質問

このオンライン画面録画ツールにはソフトウェアのインストールが必要ですか?
ソフトウェアのインストールは必要ありません。このツールはブラウザの画面キャプチャと MediaRecorder 機能を通じて実行され、記録結果はローカルに生成されます。より完全なシステム オーディオ サポートを得るには、Chrome または Edge を使用することをお勧めします。
システムサウンドを録音するにはどうすればよいですか?
「録音開始」をクリックした後、ブラウザ共有ポップアップウィンドウで画面、ウィンドウ、またはタブを選択し、「オーディオを共有する」または「システムオーディオを同時に共有する」にチェックを入れます。チェックを入れないと、録画結果が映像とマイク音声のみになる場合があります。
画面録画中にシステム音が聞こえないのはなぜですか?
最も一般的な理由は、共有ポップアップ ウィンドウでオーディオ オプションがチェックされていないことです。ブラウザまたはシステムの制限により、システム オーディオのキャプチャが妨げられる場合もあります。 Firefox はシステム オーディオのサポートが制限されており、macOS はシナリオによっては仮想サウンド カードを必要とする場合があります。
カメラとマイクを同時に録画できますか？
はい。「録画開始」をクリックする前にマイクとカメラをオンにしてください。カメラは右下に円形のピクチャーインピクチャーとして重ねられ、録画開始後はカメラの状態を変更できません。
対応しているエクスポート形式は？
このツールは現在のブラウザ対応状況に応じて利用可能な形式を検出します。一般的な選択肢は WebM VP9、WebM VP8、WebM で、一部のブラウザでは MP4 に対応する場合もあります。変換が必要な場合は、ファイルをダウンロードして 動画形式変換ツール を使用してください。
録画ファイルはサーバーにアップロードされますか?
いいえ。録画は現在のブラウザ セッションに保存され、録画リストでプレビュー、ダウンロード、または削除できます。ページを更新する前、またはタブを閉じる前に、保存しておきたい動画をダウンロードしてください。
画面録画に失敗した場合はどうすればよいですか?
ブラウザが画面キャプチャをサポートし、画面、マイク、またはカメラの権限を許可していることを確認してください。ボタンが反応しない場合、または許可ポップアップ ウィンドウが表示されない場合は、ページを更新するか、競合する画面録画ソフトウェアを閉じるか、Chrome / Edge を使用して再試行できます。
