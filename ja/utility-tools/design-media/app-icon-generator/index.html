デュアルプラットフォームのサポート

iOS は連続曲率コーナーを使用し、Android は円または正方形を使用します

iOS@1x/@2x/@3x および Android の複数の DPI サイズを含む

リアルタイム プレビュー

編集中にアイコンが iOS および Android デバイスでどのように表示されるかをリアルタイムで確認できます。

左側で編集、右側ですぐにプレビューを確認できます

一括エクスポート

ワンクリックですべてのアイコン サイズを生成し、プラットフォームとサイズごとに整理された ZIP ファイルとしてダウンロードします。

ZIP には個別の iOS フォルダーと Android フォルダーが含まれています

プライバシーとセキュリティ

純粋なフロントエンド処理 – 画像データがデバイスから流出することはなく、デザインのプライバシーを保護します。

すべての処理はブラウザ内で行われ、サーバーへのアップロードは行われません

サポートされているプラットフォームと技術仕様

iOS プラットフォームのアイコン仕様

  • • App Store: 1024×1024px (PNG、透過なし)
  • • iPhone @3x: 180×180px (Retina HD)
  • • iPhone @2x: 120×120px (Retina)
  • • iPad Pro: 167×167px
  • • iPad @2x: 152×152px
  • • iPad @1x: 76×76px
  • • システムによって自動的に適用される連続曲率コーナーマスク

Android プラットフォームのアイコン仕様

  • • Play ストア: 512×512px (PNG、32 ビット)
  • • xxxhdpi: 192×192px (超超超超高密度)
  • • xxhdpi: 144×144px (超超高密度)
  • • xhdpi: 96×96px (超高密度)
  • ・hdpi：72×72px（高密度）
  • • mdpi: 48×48px (中密度)
  • • ldpi: 36×36px (低密度)

技術的なメモ: このツールは、HTML5 Canvas テクノロジを使用してアイコンを生成し、最新のすべてのブラウザ (Chrome 60 以降、Firefox 55 以降、Safari 11 以降、Edge 79 以降) と互換性があります。

よくある質問 (FAQ)

特定のサイズでアイコンがぼやけるのはなぜですか?

アイコンがぼやけるのは、通常、ソース画像の解像度が不十分であることが原因です。

をお勧めします。
  • • 少なくとも 1024×1024 ピクセルのソース画像を使用してください
  • • ベクター (SVG) または高解像度 PNG を推奨
  • • アイコン内の主要な要素が大きく明確であることを確認してください
  • • ソース画像で過度の詳細を避けます。サイズが小さいと詳細が失われます

異なるプラットフォーム間で一貫した表示を保証するにはどうすればよいですか?

プラットフォームの違いは正常ですが、次のヒントを参考に最適化できます:

  • • 重要なコンテンツは中央の セーフ ゾーン (端を避ける) 内に保管してください
  • • 小さなサイズでもはっきりと認識できるように、ハイコントラストのデザインを使用します
  • • 右側のパネルで両方のプラットフォームのリアルタイム プレビューを確認してください
  • • iOS では丸い角が自動的に追加されます。 Android の形状はテーマによって異なります。それに応じて設計してください

透明な背景はサポートされていますか?

はい、透明な背景は完全にサポートされています:

  • • アップロードされた PNG の透明領域は保持されます
  • • 生成されたアイコンは元の透明度を維持
  • • 不透明な背景を必要とするプラットフォーム (iOS App Store など) の場合、白い背景が自動的に追加されます
  • • さまざまな背景で透明度がどのように見えるかを検討してください

どのようなファイル形式を生成できますか?

現在、アイコンは主に PNG 形式で生成されます:

  • • すべてのアイコンは PNG として生成されます (クロスプラットフォーム形式を推奨)
  • • PNG は透明な背景と高品質の圧縮をサポートしています
  • • Android の場合、WebP の方が圧縮率が高い場合がありますが、別途変換が必要です
  • • iOS App Store アイコンには透明度のない PNG が必要です

このツールは無料ですか?制限はありますか?

はい、App Icon Studio は完全に無料です:

  • • 無料で使用でき、登録やログインは必要ありません
  • • 使用制限なし
  • • 画像サイズの制限なし (ただし、画像が大きい場合は処理に時間がかかる場合があります)
  • • すべての処理はブラウザ内で行われ、サーバー費用はかかりません
  • • 将来的には高度な機能が追加される可能性がありますが、コア機能は引き続き無料です

生成されたアイコンをプロジェクトに統合するにはどうすればよいですか?

統合手順はプラットフォームによって異なります:

  • iOS (Xcode): Assets.xcassets に設定された AppIcon にアイコンをドラッグします
  • Android (Android Studio): アイコンを res/mipmap‑*dpi/ フォルダーにコピー
  • React Native: React‑native‑make ツールを使用するか、手動で ios/ および android/ ディレクトリに配置
  • Flutter: pubspec.yaml のアイコン設定を更新し、適切なディレクトリに配置します

生成された ZIP は、使いやすいようにプラットフォームとサイズごとにすでに整理されています。
