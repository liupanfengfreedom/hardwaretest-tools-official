## 🔍 检测原理简介：
- **鼠标测试**：通过JavaScript高精度时间API记录点击间隔，分析微动开关状态
- **键盘测试**：监听键盘全局事件，可视化每个按键的按下与释放状态
- **音频测试**：通过Web Audio API获取麦克风输入流，分析音频波形与频谱
- **屏幕测试**：使用全屏CSS色彩块与Canvas动态渲染，激发像素点状态显示

如果发现硬件存在问题，建议及时联系厂商进行维修或更换。定期使用我们的工具检测硬件状态，可以及早发现问题，避免影响工作和娱乐体验。
