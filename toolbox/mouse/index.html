How to use this Mouse Test tool?

This tool tests all basic functions of your mouse. Click any button (Left, Right, Middle, Side buttons) to see them light up on the visualizer. The panel on the right tracks your press and release counts accurately.

How to detect Mouse Double Clicking?

The "Double Click Bug" happens when a single physical press registers as two inputs due to switch degradation. Check the event log. If you see two presses with very short time intervals (e.g., < 40ms) for a single click, your mouse switch might be failing. The log will show a warning message for intervals less than 80ms.

Double Click Detection Feature

This tool now includes double click detection:

  • Normal Double Click: Quickly click the left mouse button twice with an interval between 80-500ms, the log will show orange text "Left Button Double Click"
  • Faulty Double Click: If the interval between two clicks is less than 80ms, it will be marked as faulty, and the log will show red text "Left Button Double Click [Faulty Double Click]"
  • Visual Feedback: When a double click occurs, the left button area will have a noticeable orange or red flashing effect

Common Troubleshooting

  • Scroll Wheel Jumps: Scroll fast and check if the counter increases smoothly.
  • Unresponsive Button: If a button does not light up, the switch may be broken.
  • Right-click displaying pressed state: In some browsers, if you press and hold the right mouse button while simultaneously moving the mouse, and then release the right mouse button, the right mouse button will still display as pressed. Therefore, it is best not to move the mouse when testing key presses.
  • Button Press/Release Count Mismatch: If press count is much higher than release count, your mouse might have button sticking issues.
  • Double Click Detection: Check the event log for time intervals between clicks. Intervals less than 80ms will be marked with a warning.
