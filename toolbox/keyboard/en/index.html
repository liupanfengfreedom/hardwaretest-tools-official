Frequently Asked Questions (FAQ)

Why can't some keys be detected?

This may be due to browser security restrictions or keyboard hardware issues. Please ensure: 1) Use modern browsers (Chrome/Firefox/Edge); 2) Focus is on the test page; 3) Check keyboard connection.

What is NKRO (N-Key Rollover)?

NKRO means the keyboard can recognize unlimited simultaneous key presses without conflicts. Regular keyboards typically support 6-key rollover, while gaming keyboards support full NKRO.

Learn more: NKRO Special Test

How to identify keyboard faults?

If certain keys: 1) Don't respond at all; 2) Have significant response delay; 3) Exhibit key chattering; 4) Conflict with other keys, it may indicate hardware issues.

Troubleshooting: Keyboard Troubleshooting Guide

What keyboard types are supported?

All keyboard types are supported: mechanical keyboards, membrane keyboards, capacitive keyboards, laptop keyboards. Compatible with Windows, macOS, Linux systems.

How to test keyboard latency?

Rapidly press the same key repeatedly and observe the Time Delta value. Professional gaming keyboards typically have 1-5ms latency, while regular keyboards may have 10-20ms.

Special Test: Keyboard Latency Special Test

Will test data be saved?

All test data is only saved in the current browser session. Data will be lost after refreshing the page or closing the tab. Use screenshot function if you need to save results.

Data Security: All data is processed locally only
