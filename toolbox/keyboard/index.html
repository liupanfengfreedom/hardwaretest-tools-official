Keyboard Test Lab - 专业键盘测试工具

这是一个专业的在线键盘测试工具，旨在帮助用户全面检测键盘的各项功能状态，包括按键响应、全键无冲性能、轴体故障等。

🔍 全键位检测

逐一测试键盘上的每个按键，确保所有键位都能正常响应。已测试的按键会显示绿色高亮，并记录击键次数。

⚡ NKRO 无冲测试

测试键盘的全键无冲(N-Key Rollover)能力。同时按下多个按键，观察"当前并发"计数，了解键盘的防冲突性能。

⏱️ 响应速度

实时显示按键时间间隔，帮助您评估键盘的响应延迟和输入流畅度。

🔧 技术参数

显示 event.key、event.code、event.which 等底层技术参数，便于开发者和高级用户进行深度分析。

🎯 使用指南

  1. 基础测试：依次按下所有按键，确保每个键都能正常触发并显示绿色
  2. 无冲测试：同时按下多个按键（建议10个以上），测试键盘的防冲突能力
  3. 快速连击：快速重复按下同一按键，测试轴体响应和防连击功能
  4. 组合键测试：测试常用组合键如 Ctrl+C、Alt+Tab 等是否正常工作

❓ 常见问题

为什么某些按键无法检测？

这可能是由于浏览器安全限制或键盘硬件问题。请确保：1) 使用现代浏览器(Chrome/Firefox/Edge)；2) 焦点在测试页面上；3) 检查键盘连接。

什么是全键无冲(NKRO)？

全键无冲指键盘可以同时识别无限多个按键输入而不会发生冲突。普通键盘通常支持6键无冲，专业游戏键盘支持全键无冲。

如何判断键盘故障？

如果某些按键：1) 完全无响应；2) 响应延迟明显；3) 出现连击现象；4) 与其他按键冲突，可能表示键盘存在硬件故障。

支持哪些类型的键盘？

支持所有类型的键盘：机械键盘、薄膜键盘、静电容键盘、笔记本键盘等。兼容 Windows、macOS、Linux 系统。
