100% தனியுரிமைப் பாதுகாப்பு

அனைத்து ஐகான் செயலாக்கமும் உங்கள் உலாவியில் உள்நாட்டில் செய்யப்படுகிறது, எந்த சேவையகத்திலும் பதிவேற்றப்படவில்லை. உங்கள் படங்கள் எப்போதும் உங்கள் சாதனத்தில் இருக்கும், இது மிக உயர்ந்த தனியுரிமை பாதுகாப்பை உறுதி செய்கிறது.

தனியுரிமை உத்தரவாதம்: வணிக லோகோக்கள் மற்றும் உணர்திறன் படங்களுக்கு ஏற்ற படக் கசிவுகளைப் பற்றி கவலைப்படத் தேவையில்லை

முழுமையான PWA கிட்

ஐசிஓ, எஸ்விஜி, ஆப்பிள் டச் ஐகான், ஆண்ட்ராய்டு குரோம் ஐகான்கள் மற்றும் வெப் மேனிஃபெஸ்ட் கோப்பு உள்ளிட்ட முழுமையான முற்போக்கு வலை பயன்பாட்டு ஐகான் கிட் ஒன்றை உருவாக்கவும், ஒரே கிளிக்கில் பயன்படுத்த தயாராக உள்ளது.

மல்டி-ரெசல்யூஷன்: ICO கோப்பில் 16×16, 32×32, 48×48 மூன்று தீர்மானங்கள் உள்ளன

நிகழ் நேர சாதன முன்னோட்டம்

iOS, Android மற்றும் Web உலாவிகளுக்கான நிகழ்நேர முன்னோட்ட விளைவுகளை வழங்குகிறது, வெவ்வேறு சாதனங்கள் மற்றும் காட்சிகளில் ஐகான் காட்சி விளைவுகளை உள்ளுணர்வுடன் பார்க்கிறது.

பல காட்சி முன்னோட்டம்: தாவல்கள், புக்மார்க்குகள், முகப்புத் திரை, தேடல் முடிவுகள் போன்றவை.

உள்ளமைக்கப்பட்ட இமேஜ் க்ராப்பிங் கருவி இழுத்தல் மற்றும் பெரிதாக்குதல் ஆகியவற்றை ஆதரிக்கிறது, துல்லியமான ஐகான் பகுதி தேர்வை உறுதிசெய்து உயர்தர சதுர சின்னங்களை உருவாக்குகிறது.

சதுர தேர்வுமுறை: ஃபேவிகான்கள் சதுரமாக இருக்க வேண்டும், கருவி தானாகவே பயிர் செய்யும் பகுதியை மேம்படுத்துகிறது

ஒரு கிளிக் தொகுப்பு பதிவிறக்கம்

உருவாக்கப்பட்ட அனைத்து ஐகான் கோப்புகளும் HTML குறியீடும் தானாகவே ஜிப் கோப்பாக தொகுக்கப்படும், இணையதள திட்டங்களில் எளிதாக ஒருங்கிணைப்பதற்கான வரிசைப்படுத்தல் வழிமுறைகளும் அடங்கும்.

பயன்படுத்தல் குறியீடு: முழு HTML குறியீடு துணுக்குகளை உள்ளடக்கியது, நகலெடுத்து பயன்படுத்த தயாராக உள்ளது

உலாவி-சொந்த தலைமுறை

ICO மற்றும் SVG கோப்புகளை உருவாக்க உலாவி-நேட்டிவ் கேன்வாஸ் API ஐப் பயன்படுத்துகிறது, சர்வர் பக்க செயலாக்கம் தேவையில்லை, வேகமான வேகம், சமீபத்திய உலாவி அம்சங்களை ஆதரிக்கிறது.

தொழில்நுட்ப ஆதரவு: HTML5 கேன்வாஸ் மற்றும் JSZip தொழில்நுட்பத்தின் அடிப்படையில், தூய முகப்புச் செயலாக்கம்

Favicon தொழில்நுட்ப விவரக்குறிப்புகள்

இந்தக் கருவியால் உருவாக்கப்பட்ட சின்னங்கள் சமீபத்திய இணையத் தரநிலைகள் மற்றும் சிறந்த நடைமுறைகளைப் பின்பற்றுகின்றன:

  • ICO வடிவம்: 16×16, 32×32, 48×48 மூன்று தீர்மானங்களைக் கொண்டுள்ளது
  • SVG வடிவம்: வெக்டார் ஐகான்கள், எல்லையற்ற அளவிடுதலுக்கான ஆதரவு, சிறிய கோப்பு அளவு
  • ஆப்பிள் டச் ஐகான்: 180×180 பிக்சல்கள், iOS முகப்புத் திரையை ஆதரிக்கிறது
  • Android Chrome ஐகான்கள்: 192×192 மற்றும் 512×512 பிக்சல்கள்
  • Web Manifest: W3C நிலையான இணக்கமான PWA மேனிஃபெஸ்ட் கோப்பு
  • HTML குறியீடு: சொற்பொருள் HTML5 தரநிலைகளைப் பின்பற்றுகிறது
  • உலாவி இணக்கத்தன்மை: குரோம், பயர்பாக்ஸ், சஃபாரி, எட்ஜ் மற்றும் பிற நவீன உலாவிகள்

தொழில்நுட்ப குறிப்பு:இந்த கருவி படங்களை உருவாக்க HTML5 கேன்வாஸ் API மற்றும் கோப்பு பேக்கேஜிங்கிற்கான JSZip நூலகத்தைப் பயன்படுத்துகிறது. உருவாக்கப்பட்ட ICO கோப்புகள் நிலையான விண்டோஸ் ஐகான் வடிவமாகும், இது அனைத்து முக்கிய உலாவிகளாலும் ஆதரிக்கப்படுகிறது. SVG வடிவம், நவீன உலாவிகளால் விரும்பப்படும், சிதைவு இல்லாமல் எல்லையற்ற அளவிடுதலுக்கான திசையன் ஆதரவை வழங்குகிறது. ஆப்பிள் டச் ஐகான் ஆப்பிளின் மனித இடைமுக வழிகாட்டுதல்களைப் பின்பற்றுகிறது, ஆண்ட்ராய்டு குரோம் ஐகான்கள் கூகிளின் PWA தரநிலைகளைப் பின்பற்றுகின்றன.

அடிக்கடி கேட்கப்படும் கேள்விகள் (FAQ)

PWA ஐகான் கிட் என்றால் என்ன? எனக்கு ஏன் அது தேவை?

A PWA (Progressive Web App) ஐகான் கிட் என்பது மொபைல் சாதனங்களில் இணையதளம் ஒரு நேட்டிவ் ஆப் போல தோற்றமளிக்க தேவையான ஐகான் கோப்புகளின் தொகுப்பாகும். இதில் பின்வருவன அடங்கும்: favicon.ico (உலாவி தாவல் ஐகான்), favicon.svg (வெக்டர் ஐகான்), apple-touch-icon.png (iOS முகப்புத் திரை ஐகான்), android-chrome-192x192.png மற்றும் android-chrome-512x512.png (Android-chrome-512x512.png (Android ஐகான்ஸ்மேன்), சிறந்த தளம். ஐகான்ஸ்மேன் கோப்பு. சிறந்த பயனர் அனுபவத்தை வழங்க நவீன இணையதளங்களுக்கு முழுமையான PWA ஐகான் கிட் தேவை.

எனக்கு ஏன் SVG வடிவ ஃபேவிகான் தேவை?

SVG (அளவிடக்கூடிய வெக்டர் கிராபிக்ஸ்) வடிவம் ஃபேவிகானில் பின்வரும் நன்மைகள் உள்ளன: 1) திசையன் வடிவம், சிதைவு இல்லாமல் எல்லையற்ற அளவிடுதல், பல்வேறு தெளிவுத்திறன் சாதனங்களுக்கு ஏற்ப; 2) சிறிய கோப்பு அளவு, வேகமாக ஏற்றுதல்; 3) CSS ஸ்டைலிங் மற்றும் அனிமேஷன் விளைவுகளை ஆதரிக்கிறது; 4) நவீன உலாவிகள் (Chrome, Firefox, Safari, Edge) அனைத்தும் SVG ஃபேவிகானை ஆதரிக்கின்றன. பழைய உலாவிகளுக்கு ஐசிஓ வடிவமைப்பை வழங்கும் அதே வேளையில் நவீன இணையதளங்கள் SVG வடிவமைப்பிற்கு முன்னுரிமை அளிக்க பரிந்துரைக்கப்படுகிறது.

உருவாக்கப்பட்ட ICO கோப்புகள் ஏன் பல தீர்மானங்களைக் கொண்டிருக்கின்றன?

ICO கோப்பு வடிவம் "மல்டி-ரெசல்யூஷன் ICO" எனப்படும் ஒரு கோப்பில் பல தெளிவுத்திறன் படங்களை உட்பொதிப்பதை ஆதரிக்கிறது. உலாவிகள் மற்றும் இயக்க முறைமைகள் பயன்பாட்டு சூழ்நிலையின் அடிப்படையில் பொருத்தமான தெளிவுத்திறனைத் தானாகவே தேர்ந்தெடுக்கும்: உலாவி தாவல்களுக்கு 16×16 பிக்சல்கள், பணிப்பட்டிகள் மற்றும் பிடித்தவைகளுக்கு 32×32 பிக்சல்கள், டெஸ்க்டாப் குறுக்குவழிகளுக்கு 48×48 பிக்சல்கள். பல தனித்தனி கோப்புகளை விட ஒற்றை மல்டி-ரெசல்யூஷன் ICO கோப்பு மிகவும் திறமையானது, HTTP கோரிக்கைகளை குறைக்கிறது.

எனது இணையதளத்தில் உருவாக்கப்படும் ஐகான்களை எவ்வாறு வரிசைப்படுத்துவது?

பதிவிறக்கம் செய்யப்பட்ட ZIP தொகுப்பில் வரிசைப்படுத்தல் வழிமுறைகள் மற்றும் குறியீடு துணுக்குகள் உள்ளன. அடிப்படை படிகள்: 1) அனைத்து ஐகான் கோப்புகளையும் (.ico, .svg, .png) உங்கள் வலைத்தளத்தின் ரூட் கோப்பகத்தில் பதிவேற்றவும், 2) மேலும் site.webmanifest கோப்பை ரூட் கோப்பகத்தில் பதிவேற்றவும், 3) உங்கள் HTML கோப்பின் <head> பிரிவில் வழங்கப்பட்ட குறியீடு துணுக்கைச் சேர்க்கவும். பெரும்பாலான சந்தர்ப்பங்களில், நீங்கள் குறியீட்டை நகலெடுத்து ஒட்ட வேண்டும். கருவி உருவாக்கிய குறியீடு ஏற்கனவே SVG மற்றும் ICO முன்னுரிமை அமைப்புகள் உட்பட தேவையான அனைத்து இணைப்புகள் மற்றும் மெட்டா குறிச்சொற்களை உள்ளடக்கியது.

IOS முகப்புத் திரையில் தனிப்பயன் ஐகான்களை எனது இணையதளம் ஏன் காட்டவில்லை?

iOS முகப்புத் திரையில் தனிப்பயன் ஐகானுக்குப் பதிலாக வலைப்பக்கத்தின் ஸ்கிரீன் ஷாட்டை உங்கள் இணையதளம் காட்டினால், சாத்தியமான காரணங்கள்: 1) ஆப்பிள் டச் ஐகான் கோப்பு ரூட் டைரக்டரியில் சரியாக வைக்கப்படவில்லை, 2) HTML குறியீட்டில் தவறான ஆப்பிள்-டச்-ஐகான் இணைப்பு, 3) iOS Safari பழைய ஐகானைத் தேக்ககப்படுத்தியது. எல்லா கோப்புகளும் சரியாகப் பயன்படுத்தப்பட்டுள்ளதை உறுதிசெய்து, சஃபாரியின் இணையதளத் தரவை அழிக்க முயற்சிக்கவும். இந்தக் கருவியால் உருவாக்கப்பட்ட குறியீடு ஏற்கனவே தேவையான அனைத்து apple-touch-icon இணைப்புகளையும் கொண்டுள்ளது.

ஃபேவிகான்களுக்கான சிறந்த வடிவமைப்பு நடைமுறைகள் யாவை?

ஃபேவிகான்களை வடிவமைக்கும் போது, ​​நாங்கள் பரிந்துரைக்கிறோம்: 1) எளிய கிராபிக்ஸ் அல்லது எழுத்துக்களைப் பயன்படுத்தவும், சிக்கலான விவரங்களைத் தவிர்க்கவும் (அளவு சிறியது), 2) படிக்கக்கூடிய உயர் மாறுபட்ட வண்ணங்களைப் பயன்படுத்தவும், 3) சதுர வடிவமைப்பைப் பராமரிக்கவும் (சதுரப் பின்னணியில் வட்ட வடிவ வடிவமைப்புகள்), 4) மெல்லிய கோடுகளைத் தவிர்க்கவும் (வெவ்வேறு பின்னணி வண்ணங்களில் தெரியாமல் இருக்கலாம்), 5) இந்த கருவியின் நிகழ்நேர முன்னோட்ட அம்சம் வடிவமைப்பு விளைவுகளை மதிப்பிட உதவும்.

இந்தக் கருவி இலவசமா?

ஆம், Favicon Studio Pro என்பது முற்றிலும் இலவச ஆன்லைன் கருவியாகும், பதிவு அல்லது உள்நுழைவு தேவையில்லை. டெவலப்பர்கள் மற்றும் வடிவமைப்பாளர்களுக்கான உயர்தர வடிவமைப்புக் கருவிகளை வழங்குவதற்கு நாங்கள் உறுதிபூண்டுள்ளோம், தொழில்முறை இணையதள ஐகான் கிட்களை விரைவாக உருவாக்க உங்களுக்கு உதவுகிறோம். அனைத்து செயலாக்கங்களும் உங்கள் உலாவியில் உள்நாட்டில் செய்யப்படுகின்றன, உங்கள் படங்கள் எந்த சேவையகத்திலும் பதிவேற்றப்படாது.
