மல்டி-ப்ரோட்டோகால் (WS/WSS)

ஒரே கிளிக்கில் ws மற்றும் wss இடையே மாறவும். தானாக-முழுமையான URL முன்னொட்டுகள், மறைகுறியாக்கப்பட்ட மற்றும் எளிய இணைப்புகளுடன் இணக்கமானது.

அடிக்கடி கேட்கப்படும் கேள்விகள் (FAQ)

WebSocket என்றால் என்ன?

WebSocket என்பது ஒரு TCP இணைப்பில் முழு-இரட்டைத் தொடர்பை வழங்கும் ஒரு நெறிமுறையாகும், இது சேவையகங்களை வாடிக்கையாளர்களுக்கு தரவை அனுப்ப அனுமதிக்கிறது. இந்த கருவி உங்கள் WebSocket சேவை சரியாக வேலை செய்கிறது என்பதை சோதிக்க உதவுகிறது.

WebSocket இணைப்பை எவ்வாறு சோதிப்பது?

WebSocket சர்வர் URL ஐ உள்ளிடவும் (எ.கா. wss://echo.websocket.org), 'இணை' என்பதைக் கிளிக் செய்யவும். இணைக்கப்பட்டதும், உள்ளீடு பகுதியில் ஒரு செய்தியைத் தட்டச்சு செய்து 'அனுப்பு' என்பதைக் கிளிக் செய்யவும். பெறப்பட்ட செய்திகள் பதிவில் தோன்றும்.

இணைப்பு ஏன் தோல்வியடைகிறது?

சாத்தியமான காரணங்கள்: தவறான சேவையக முகவரி, சேவையகம் WSS ஐ ஆதரிக்காது (WS ஐ முயற்சிக்கவும்), ஃபயர்வால் தடுப்பு, சேவையகம் செயலிழக்க அல்லது CORS கட்டுப்பாடுகள். URL ஐச் சரிபார்த்து, சர்வர் WebSocket இணைப்புகளை அனுமதிக்கிறது என்பதை உறுதிப்படுத்தவும்.

தனிப்பயன் தலைப்புகளை எவ்வாறு சேர்ப்பது?

'இணைப்பு அமைப்புகள்' என்பதன் கீழ், தலைப்பு பெயர் மற்றும் மதிப்பை உள்ளிட '+ சேர்' என்பதைக் கிளிக் செய்யவும். இந்த தலைப்புகள் WebSocket ஹேண்ட்ஷேக்கின் போது அனுப்பப்படும், அங்கீகாரம் அல்லது தனிப்பயன் அளவுருக்களுக்கு பயனுள்ளதாக இருக்கும்.

என்ன செய்தி வடிவங்கள் ஆதரிக்கப்படுகின்றன?

உரை, JSON மற்றும் ஹெக்ஸ். வடிவமைப்பைத் தேர்ந்தெடுத்து, உங்கள் உள்ளடக்கத்தை உள்ளிடவும், அது தானாகவே மாற்றப்படும். JSON பயன்முறையில் எளிதாகத் திருத்துவதற்கான வடிவமைப்பு மற்றும் சிறிய பொத்தான்கள் உள்ளன.

நான் பதிவைச் சேமிக்க முடியுமா?

ஆம். தற்போதைய பதிவை உரைக் கோப்பாகப் பதிவிறக்க, பதிவுப் பகுதியில் உள்ள 'ஏற்றுமதி' பொத்தானைக் கிளிக் செய்யவும்.

தாமதம் எவ்வாறு கணக்கிடப்படுகிறது?

இணைப்புக்குப் பிறகு, கருவியானது ஒவ்வொரு 10 வினாடிகளுக்கும் ஒரு உள்ளமைக்கப்பட்ட பிங் செய்தியை அனுப்புகிறது மற்றும் தற்போதைய தாமதத்தைக் காட்டும் பாங் பெறும் வரையிலான நேரத்தை அளவிடும்.
