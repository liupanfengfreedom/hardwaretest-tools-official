在线录屏工具使用说明

这个在线屏幕录制工具适合录制网页演示、产品操作、课程讲解、会议片段和问题复现视频。录制过程基于浏览器的屏幕捕获能力完成，视频片段会在当前设备本地生成，可直接预览、下载或删除。

开始录制前，可以先选择画质、帧率和导出格式；如果需要讲解声音，先开启麦克风；如果需要真人出镜，先开启摄像头，工具会把摄像头画面叠加到录制视频右下角。

浏览器共享弹窗详解

点击「开始录制」后，Chrome 或 Edge 会弹出屏幕共享选择窗口。你可以选择录制整个屏幕、某个应用窗口或浏览器标签页。若要录制系统声音，关键是勾选弹窗底部的「分享音频」或「同时分享系统音频」。

常见问题解答

这个在线录屏工具需要安装软件吗？
不需要安装软件。工具通过浏览器的屏幕捕获和 MediaRecorder 能力运行，录制结果在本地生成。建议使用 Chrome 或 Edge 获得更完整的系统音频支持。
如何录制系统声音？
点击「开始录制」后，在浏览器共享弹窗中选择屏幕、窗口或标签页，并勾选「分享音频」或「同时分享系统音频」。如果没有勾选，录制结果可能只有画面和麦克风声音。
为什么录屏没有系统声音？
最常见原因是共享弹窗中没有勾选音频选项。也可能是浏览器或系统限制导致系统音频无法被捕获。Firefox 对系统音频支持有限，macOS 某些场景可能需要虚拟声卡配合。
可以同时录制摄像头和麦克风吗？
可以。请在点击「开始录制」之前开启麦克风和摄像头。摄像头会以圆形画中画叠加到录制画面右下角，录制开始后暂不支持切换摄像头状态。
支持哪些导出格式？
工具会根据当前浏览器自动检测可用格式，常见选项包括 WebM VP9、WebM VP8 和 WebM；部分浏览器可能支持 MP4。若需要进一步转换，可下载后使用 视频格式转换器 处理。
录制文件会上传到服务器吗？
不会。录制片段保存在当前浏览器会话中，你可以在录制列表里预览、下载或删除。刷新页面或关闭标签页前，请先下载需要保留的视频。
录屏失败应该怎么办？
请确认浏览器支持屏幕捕获功能，并允许屏幕、麦克风或摄像头权限。若按钮无响应或权限弹窗未出现，可以刷新页面、关闭冲突的录屏软件，或换用 Chrome / Edge 重试。
