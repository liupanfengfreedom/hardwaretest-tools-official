常见问题解答 (FAQ)

支持哪些音频格式转换？

目前支持以下格式互转：MP3、WAV、FLAC、AAC (M4A)、OGG (Opus)、AIFF、WMA、WebM (Opus)。同时可以从视频文件（MP4、MKV、MOV、AVI、3GP、TS）中提取音频并转换为上述格式。未来会持续增加更多格式支持。

转换后的音频质量如何？

质量完全取决于您选择的设置：
- 若选择 WAV 或 AIFF，则为无损 PCM，与原始音频数据完全一致。
- 若选择 MP3，您可以通过滑块调整比特率（64 kbps 到 320 kbps），越高音质越好，文件越大。
- 采样率选项允许您降采样以减小文件体积（例如将 48kHz 降为 44.1kHz 或 22kHz）。
建议保留原始采样率和立体声以获得最佳效果。

文件真的不会上传吗？

是的，100% 本地处理。我们利用 Web Audio API 解码音频，使用 lamejs 编码 MP3，MediaRecorder 编码 Opus，所有操作都在您的浏览器内存中进行，没有网络传输。您甚至可以在离线状态下使用本工具（页面需提前加载）。

为什么我无法转换某些视频文件？

浏览器只能解码它支持的视频编码格式。例如，某些 MKV 文件可能包含 HEVC (H.265) 视频流，而 Safari 或 Chrome 可能不支持解码其音频。建议使用更通用的编码（如 H.264 + AAC）的视频文件。如果问题持续，您可以尝试使用格式转换工具先转换视频编码，再提取音轨。

我可以同时转换多个文件吗？

可以。您可以选择多个文件，它们会出现在文件队列中。点击“开始转换”后，工具将逐个处理所有文件，并在每个文件完成后提供下载。您可以在转换过程中继续添加或移除文件，但正在转换的文件无法修改。

转换大文件时页面会卡住吗？

音频编码是计算密集型任务，尤其是大文件或高比特率。虽然我们尽量优化，但浏览器主线程可能会短暂阻塞。对于超大文件（例如超过 100 MB），建议耐心等待，进度条会实时更新。未来版本考虑引入 Web Worker 以提升流畅度。
