年利率怎么算？

年利率计算公式：年利率 = (利息 / 本金) / 贷款年限 × 100%。例如贷款100万，20年总利息40万，年利率= (40/100)/20×100% = 2%。您可直接使用本计算器验证。

公积金贷款计算器如何使用？

输入贷款金额、公积金利率(如3.1%)、期限，选择还款方式即可查看月供、总利息。本工具完全支持公积金贷款与组合贷款分析。

房贷利率计算器准确吗？

采用银行级等额本息/本金公式，结果与专业系统一致。可自由调整LPR或固定利率，动态模拟每月还款额。

等额本息和等额本金区别？

等额本息月供固定，前期利息多；等额本金月供递减，总利息少。趋势图可直观对比两者利息分布。

定期利息怎么算？也适用于贷款吗？

存款定期利息一般用单利：利息=本金×年利率×年限。贷款则常用复利（等额本息）或单利（先息后本）。本计算器支持各类还款方式，也可辅助理解定期利息概念。
