🚀 أداة النشر عبر الإنترنت / اختبار واجهة برمجة التطبيقات

أداة تصحيح أخطاء طلب HTTP للواجهة الأمامية الخالصة، تدعم أساليب HTTP المتعددة وتنسيقات البيانات

طلب التكوين

نتيجة الاستجابة























ميزات الأداة






تنفيذ الواجهة الأمامية النقية


يعمل بشكل كامل في المتصفح، دون الحاجة إلى خادم. يتم إرسال جميع طلبات HTTP من خلال Fetch API للمتصفح، مما يضمن خصوصية البيانات وأمانها.






طرق HTTP متعددة


يدعم طرق POST وGET وPUT وDELETE وPATCH لخمس طرق HTTP، مما يلبي احتياجات تطوير RESTful API الأساسية.






دعم البيانات متعددة التنسيقات


يدعم JSON وتنسيقات بيانات النموذج، وهو مناسب لاختبار أنواع مختلفة من واجهات API. يوفر محرر JSON تسليط الضوء على بناء الجملة والتحقق من صحتها.






رؤوس الطلبات المخصصة


تكوين رؤوس الطلب بمرونة، ويدعم الرؤوس الشائعة مثل نوع المحتوى، والتفويض، ووكيل المستخدم، وتلبية متطلبات اختبار واجهة برمجة التطبيقات المعقدة.
أداة






سجل محفوظات URL


يحفظ تلقائيًا عناوين URL المستخدمة مؤخرًا، وهو مناسب للتبديل السريع والاختبار المتكرر، مما يحسن كفاءة تصحيح الأخطاء.






معاينة الاستجابة في الوقت الحقيقي


يعرض على الفور رمز حالة HTTP ورؤوس الاستجابة ونص الاستجابة، ويدعم العرض بتنسيق JSON، ويسهل تحليل نتائج إرجاع API.








الأسئلة المتداولة




هل تدعم هذه الأداة HTTPS؟


نعم، يدعم بروتوكول HTTPS بشكل كامل. يمكنك إدخال أي عنوان URL صالح لـ HTTPS للاختبار. تتبع الأداة سياسات أمان المتصفح.






كيفية التعامل مع الطلبات عبر الأصل؟


تتطلب الطلبات عبر الأصل أن يقوم الخادم الهدف بتكوين رؤوس CORS (مشاركة الموارد عبر الأصل). إذا واجهت أخطاء عبر الأصل، فتأكد من أن خادم API قام بتكوين سياسات CORS الصحيحة، مثل Access-Control-Allow-Origin وما إلى ذلك.






هل يدعم تحميل الملفات؟


الإصدار الحالي يدعم بشكل أساسي JSON وتنسيقات بيانات النموذج. سيتم دعم تنسيق بيانات النموذج/الأجزاء المتعددة (لتحميل الملفات) في الإصدارات المستقبلية. يمكنك استخدام FormData API مع هذه الأداة لاختبار تحميل الملفات.






هل يتم حفظ البيانات على الخادم؟


لا. يتم حفظ جميع البيانات في وحدة التخزين المحلية للمتصفح الخاص بك (التخزين المحلي)، ولا يتم تحميلها على أي خادم، مما يضمن مفاتيح API الخاصة بك وأمن البيانات الحساسة.






هل يدعم اختبار WebSocket؟


يركز الإصدار الحالي على اختبار طلب HTTP/HTTPS. أداة اختبار WebSocket قيد التطوير وسيتم توفيرها في الإصدارات المستقبلية.






هل هو مناسب لتعلم بروتوكول HTTP؟


مثالي للتعلم! تعرض هذه الأداة بشكل حدسي جميع مكونات طلبات واستجابات HTTP، وهي أداة مثالية لتعلم بروتوكول HTTP وتصميم RESTful API وتقنيات تصحيح الأخطاء.
