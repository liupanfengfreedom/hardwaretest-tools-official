Why exclude certain characters?

Excluding characters improves password usability:
1) Avoid confusion: like number 0 and letter O, number 1 and letter l, number 5 and letter S
2) Special scenarios: some systems may not support specific special symbols
3) Ease of typing: certain symbols are difficult to input on mobile devices
Note: excluding characters reduces character set size, which may lower password security.

Are the generated passwords secure?

Yes, CyberPass Pro uses the browser's built-in Crypto.getRandomValues() API for cryptographically secure random number generation. All operations are performed client-side; passwords are not sent to any server. As long as you choose a sufficiently large character set, generated passwords are secure.

How to save generated passwords?

We recommend using professional password managers (like Bitwarden, 1Password, LastPass, etc.) to securely store generated passwords. Do not save passwords in plain text files or share through insecure channels. This tool provides copy and export functions for easy password saving to password managers.
