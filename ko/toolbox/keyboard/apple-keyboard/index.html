Apple Magic Keyboard 및 Apple 키보드 테스터 기능

전체 키보드 시각화 · Magic Keyboard 호환

완전한 Apple ISO 키보드 레이아웃. Apple Magic Keyboard, Magic Keyboard 또는 AppleBook 내장 키보드 중 무엇을 사용하든 키를 누를 때마다 해당 키가 즉시 강조 표시됩니다. 직관적인 피드백을 위해 기능 키, 수정자 및 국제 키를 지원합니다.

정확한 채팅 감지

키를 누를 때마다 자동으로 타임스탬프를 찍습니다. 80ms 내에 두 번 누르면 로그 항목이 빨간색으로 강조 표시되어 Magic Keyboard의 잠재적인 스위치 바운스 문제를 빠르게 식별하는 데 도움이 됩니다.

팁: 일반 키 간격은 일반적으로 100ms를 초과합니다. 비정상적으로 빠른 채터링은 스위치 노후화로 인해 발생하는 경우가 많습니다.

수정자 및 콤보 키 테스트

Control, Option, Command, Fn 및 기타 수정자를 개별적으로 테스트합니다. 여러 키를 동시에 눌러 NKRO(최대 동시 키 표시)를 테스트합니다. Magic Keyboard의 나비식 또는 가위식 메커니즘을 평가하는 데 적합합니다.

실시간 로그 및 키코드 분석

모든 보도 자료는 정확한 시간, 키, 코드 및 값과 함께 기록됩니다. 이는 Magic Keyboard의 입력 세부 정보 또는 Bluetooth 대기 시간을 분석하는 개발자나 고급 사용자에게 이상적입니다.

모든 최신 브라우저와 호환 가능

바닐라 JavaScript로 제작되었으며 플러그인이 필요하지 않습니다. Chrome, Edge, Safari, Firefox에서 원활하게 실행됩니다. Bluetooth로 연결된 Magic Keyboard에서도 작동합니다.

원클릭 재설정, 반복 테스트

모든 카운트와 로그를 지우려면 "테스트 재설정"을 클릭하여 여러 테스트 라운드를 쉽게 실행하거나 다양한 키보드(예: Magic Keyboard와 타사)를 비교할 수 있습니다.

자주 묻는 질문(FAQ)

Apple Magic Keyboard의 키를 어떻게 테스트합니까?

페이지 중앙의 키보드 레이아웃을 보면서 Magic Keyboard의 아무 키나 누르세요. 해당 키가 강조 표시되고, 상태 패널에 키 이름, KeyCode 및 값이 표시되며, 로그 영역에는 타임스탬프와 함께 누르고 놓을 때마다 기록됩니다. 키가 응답하지 않으면 하드웨어 문제 또는 Bluetooth 연결 문제를 나타낼 수 있습니다.

채팅 감지란 무엇인가요? Magic Keyboard 키가 떨리는지 어떻게 알 수 있나요?

채터 감지는 동일한 키를 두 번 누르는 사이의 시간 간격을 모니터링합니다. 간격이 80밀리초 미만인 경우(일반적으로 비정상적으로 빠른 두 번 클릭을 나타냄) 로그 항목이 빨간색으로 강조 표시되고 채터 경고 레이블이 지정됩니다. 이는 일부 키보드 메커니즘에서 흔히 발생하는 잠재적인 스위치 바운스 문제를 식별하는 데 도움이 됩니다.

Apple Magic Keyboard가 키를 눌렀을 때 로그에 응답이 표시되지 않는 이유는 무엇입니까?

테스트 영역이 활성화되어 있는지 확인하십시오(아무 키나 직접 누르십시오). 여전히 응답이 없으면 브라우저 권한 문제이거나 하드웨어 문제일 수 있습니다. 페이지를 새로 고치거나 다른 브라우저를 사용해 보세요(Chrome, Edge 권장). 또한 OS가 특히 Bluetooth 연결의 경우 키보드를 인식하는지 확인하십시오. 유선 매직 키보드도 지원됩니다.

수정자 키(명령, 옵션 등)를 테스트하려면 어떻게 해야 하나요?

수정자 키(예: 왼쪽 Command, 오른쪽 Option)를 누르기만 하면 됩니다. 화면 키보드의 해당 키에 불이 들어오고 상태 패널에 키 이름과 코드가 표시됩니다. 여러 수정자를 동시에 눌러 조합 키 인식을 테스트할 수 있습니다.

어떤 Apple 키보드 레이아웃이 지원되나요? Apple Magic Keyboard가 호환됩니까?

이 도구는 기능 키(F1-F12), 숫자 키, 문자, 구두점, 맨 아래 행 수정자(Control, Option, Command) 및 화살표 키를 포함한 모든 주요 키를 포함하는 표준 ISO Apple 키보드 레이아웃을 사용합니다. Apple Magic Keyboard(숫자 키패드 버전 포함), Magic Keyboard, AppleBook 내장 키보드 및 기타 타사 Apple 키보드와 완벽하게 호환됩니다.

Magic Keyboard의 NKRO(최대 동시 키)의 의미는 무엇입니까?

최대 동시 키는 동시에 눌러도 올바르게 인식될 수 있는 키 수를 나타냅니다. 게이머나 복잡한 단축키를 사용하는 사용자에게는 높은 NKRO 값이 중요합니다. Magic Keyboard는 일반적으로 6키 롤오버를 지원합니다. 이 도구는 현재 누르고 있는 키 수를 기록하고 테스트 중 기록 최대값을 표시하여 키보드의 실제 동시성 기능에 대한 통찰력을 제공합니다.
