마우스 테스트 기능

왼쪽/오른쪽 클릭, 스크롤 휠, DPI 전환 및 측면 버튼을 포함한 모든 마우스 기능에 대한 포괄적인 테스트입니다. 일반적인 더블클릭 문제를 전문적으로 감지합니다.

  • 왼쪽/오른쪽 클릭 반응 테스트
  • 스크롤 휠 감도 및 방향 감지
  • DPI 스위칭 기능 검증
  • 측면 버튼 기능 확인

키보드 테스트 기능

키 충돌 감지, 키 채터링 분석, 기능 키 검증을 포함한 전체 키 기능 테스트입니다. 다양한 기계식 및 멤브레인 키보드 레이아웃을 지원합니다.

  • 전체 키 기능 테스트
  • 키 충돌 감지
  • 주요 수다분석
  • 기능키 및 조합 확인

카메라 테스트 기능

해상도, 프레임 속도, 자동 초점 및 색상 정확도를 포함한 포괄적인 카메라 상태 감지. 전면 및 후면 카메라 전환 테스트를 지원합니다.

  • 해상도 및 이미지 품질 감지
  • 프레임 속도 및 부드러움 테스트
  • 오토포커스 기능 검증
  • 색상 정확도 분석

마이크 테스트 기능

민감도 테스트, 소음 분석, 오디오 품질 평가를 포함한 정확한 마이크 상태 감지. 실시간 파형 시각화를 제공합니다.

  • 마이크 감도 테스트
  • 배경 소음 분석
  • 오디오 품질 평가
  • 실시간 파형 시각화

화면 데드 픽셀 감지 기능

밝은 점, 데드 픽셀, 빛 누출 및 고르지 않은 색상을 식별하는 데 사용할 수 있는 전문적인 전체 화면 단색 감지 모드를 제공합니다. 간섭 방지 및 흔들림 방지 설계와 결합된 다양한 색상 전환 모드를 지원하여 안정적이고 신뢰할 수 있는 테스트를 보장합니다.

  • 전체 화면 단색 데드 픽셀 감지
  • 8가지 색상 전환 옵션 제공
  • 테스트 모드에서는 마우스와 인터페이스가 자동으로 숨겨집니다.
  • ESC로 종료하고 인터페이스 자동 복원 지원
