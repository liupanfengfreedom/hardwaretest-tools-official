macOS 시스템 오디오에는 BlackHole 가상 드라이버가 필요합니다. 그런 다음 공유 대화상자에서 해당 장치를 선택하세요.
Firefox 시스템 오디오는 지원되지 않습니다(마이크 작동). 전체 기능을 이용하려면 Chrome/Edge를 사용하세요.

공유 취소

단계:

  1. 공유할 항목을 선택하세요(모든 시스템 오디오를 캡처하려면 '전체 화면' 선택).
  2. 하단의 "시스템 오디오 공유" 상자 를 선택합니다(표현은 다를 수 있음).
  3. 녹화를 시작하려면 "공유"를 클릭하세요.

⚠️ 오디오 확인란이 없으면 소리가 나지 않거나 녹음이 실패합니다.

자주 묻는 질문

MP3를 어떻게 녹음하나요?
현재 이 도구는 WEBM(원본 품질) 및 WAV(무손실)로 내보내기를 지원합니다. MP3 형식이 필요한 경우 먼저 WAV로 내보낸 다음 오디오 변환기 도구 을 사용하여 변환하는 것이 좋습니다. 향후 버전에서는 직접 MP3 내보내기를 지원할 계획입니다. 계속 지켜봐 주시기 바랍니다!
시스템 오디오를 어떻게 녹음하나요?
소스로 '시스템 오디오'를 선택하고 녹화를 클릭하세요. 브라우저 공유 대화 상자에서 공유 대상(예: 전체 화면)을 선택하고 하단의 "시스템 오디오 공유"를 확인하세요. 시작하려면 확인하세요.
시스템 사운드를 캡처할 수 없는 이유는 무엇입니까?
팝업에서 '오디오 공유' 확인란을 선택하는 것을 잊었을 가능성이 높습니다. 또한 macOS에는 BlackHole이 설치되어 있어야 합니다. Firefox는 시스템 오디오를 지원하지 않습니다. Chrome/Edge를 사용하세요.
공유 대화상자에는 어떤 옵션이 표시되나요?
'크롬 탭', '창', '전체 화면'의 세 가지 탭이 표시됩니다. 원하는 것을 선택하되 하단의 "시스템 오디오 공유" 상자를 선택한 다음 공유를 클릭해야 합니다.
어떤 내보내기 형식을 사용할 수 있나요?
설정에서 자동 내보내기(WEBM(원본) 또는 WAV(무손실))를 선택할 수 있습니다. 자동 내보내기가 없으면 녹음은 나중에 수동으로 내보내기 위해 목록에 유지됩니다.
예정정차는 어떻게 이용하나요?
"예정된 중지"에 숫자를 입력하고 단위(초/분/시간)를 선택하세요. 예: 5분 이상 = 5분 후에 중지됩니다. 0 또는 비어 있음 = 수동 중지.
