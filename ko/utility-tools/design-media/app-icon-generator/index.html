지원되는 플랫폼 및 기술 사양

iOS 플랫폼 아이콘 사양

  • • App Store: 1024×1024px(PNG, 투명도 없음)
  • • iPhone @3x: 180×180px(Retina HD)
  • • iPhone @2x: 120×120px(레티나)
  • • 아이패드 프로: 167×167px
  • • iPad @2x: 152×152px
  • • iPad @1x: 76×76px
  • •
    • 시스템에 의해 자동으로 적용되는 연속 곡률 코너 마스크

Android 플랫폼 아이콘 사양

  • • Play 스토어: 512×512px(PNG, 32비트)
  • • xxxhdpi: 192×192px(초초초고밀도)
  • • xxhdpi: 144×144px(초초고밀도)
  • • xhdpi: 96×96px(초고밀도)
  • • hdpi: 72×72px(고밀도)
  • • mdpi: 48×48px(중밀도)
  • • ldpi: 36×36px(저밀도)

기술 노트: 이 도구는 HTML5 Canvas 기술을 사용하여 모든 최신 브라우저(Chrome 60+, Firefox 55+, Safari 11+, Edge 79+)와 호환되는 아이콘을 생성합니다.

자주 묻는 질문(FAQ)

내 아이콘이 특정 크기에서 흐릿해지는 이유는 무엇입니까?

흐릿한 아이콘은 일반적으로 소스 이미지 해상도가 부족하기 때문에 발생합니다. 권장사항:

  • • 최소 1024×1024픽셀
    • 의 소스 이미지를 사용하세요.
  • • 벡터(SVG) 또는 고해상도 PNG 선호
  • • 아이콘의 핵심 요소가 크고 명확한지 확인하세요
  • • 소스 이미지의 과도한 세부 묘사는 피하세요. 작은 크기에서는 세부 정보가 손실됩니다

다양한 플랫폼에서 일관된 표시를 보장하려면 어떻게 해야 합니까?

플랫폼 차이는 일반적이지만 다음 팁을 사용하여 최적화할 수 있습니다.

  • • 중요한 콘텐츠를 중앙 안전 지대 (가장자리를 피함)
    • 안에 보관하세요.
  • • 작은 크기에서도 선명한 인식을 위해 고대비 디자인을 사용합니다
  • • 오른쪽 패널에서 두 플랫폼의 실시간 미리보기를 확인하세요
  • • iOS는 둥근 모서리를 자동으로 추가합니다. Android 모양은 테마에 따라 다양하므로 그에 맞게 디자인하세요

투명한 배경을 지원하나요?

예, 투명한 배경이 완벽하게 지원됩니다.

  • • 업로드된 PNG의 투명한 영역은
    • 유지됩니다.
  • • 생성된 아이콘은 원본 투명성을 유지합니다
  • • 불투명한 배경이 필요한 플랫폼(예: iOS App Store)의 경우 흰색 배경이 자동으로 추가됩니다
  • • 다양한 배경에서 투명성이 어떻게 보이는지 고려하세요

어떤 파일 형식을 생성할 수 있나요?

현재 아이콘은 주로 PNG 형식으로 생성됩니다:

  • • 모든 아이콘은 PNG로 생성됩니다(크로스 플랫폼 형식 권장)
  • • PNG는 투명한 배경과 고품질 압축을 지원합니다
  • • Android의 경우 WebP는 더 나은 압축을 제공할 수 있지만 별도의 변환이 필요합니다
  • • iOS App Store 아이콘에는 투명도가 없는 PNG가 필요합니다

이 도구는 무료인가요? 제한사항이 있나요?

네, App Icon Studio는 완전 무료입니다:

  • • 무료로 사용할 수 있으며 등록이나 로그인이 필요하지 않습니다
  • • 사용 제한 없음
  • • 이미지 크기 제한 없음(큰 이미지는 처리하는 데 시간이 더 오래 걸릴 수 있음)
  • • 모든 처리는 브라우저 내에서 이루어지며 서버 비용은 없습니다
  • • 향후 고급 기능이 추가될 수 있지만 핵심 기능은 무료로 유지됩니다

생성된 아이콘을 내 프로젝트에 어떻게 통합하나요?

통합 단계는 플랫폼에 따라 다릅니다:

  • iOS(Xcode): Assets.xcassets
    • 에 설정된 AppIcon으로 아이콘을 드래그합니다.
  • Android(Android Studio): res/mipmap‑*dpi/ 폴더에 아이콘 복사
  • React Native: React-native-make 도구를 사용하거나 ios/ 및 android/ 디렉토리에 수동으로 배치합니다
  • Flutter: pubspec.yaml의 아이콘 구성을 업데이트하고 적절한 디렉터리에 배치합니다

생성된 ZIP은 이미 플랫폼별, 크기별로 정리되어 있어 쉽게 사용할 수 있습니다.
