전문 오디오 변환기 기능

🎯 무손실 및 손실 지원

WAV, AIFF 무손실 형식 또는 MP3, OGG, WebM 손실 형식 중에서 선택하세요. 보관, 공유 또는 모바일 장치에 적합합니다.

최대 320kbps의 MP3 비트 전송률, 무손실에 가까운 품질.

🎬 비디오에서 오디오 추출

MP4, MKV, MOV, AVI 파일을 드래그하면 선택한 형식으로 오디오 스트림이 자동으로 추출됩니다. 추가 도구가 필요하지 않습니다.

일반적인 비디오 형식을 지원합니다. 디코딩에 실패하면 먼저 비디오를 다시 인코딩해 보십시오.

⚙️ 미세한 매개변수 제어

샘플 속도(8kHz ~ 96kHz), 채널 수(모노/스테레오) 및 비트 전송률(MP3)을 사용자 정의하세요. 출력 품질을 완벽하게 제어할 수 있습니다.

원본 파일의 샘플링 속도와 채널을 유지하려면 "자동"을 선택하세요.

🔒 개인정보 보호, 업로드 없음

모든 처리는 브라우저 내에서 이루어집니다. 귀하의 파일은 절대로 장치를 떠나지 않습니다. 민감한 녹음을 자신있게 변환하세요.

개인 정보 보호 걱정은 없습니다. 변환하고 삭제하세요.

📦 일괄 변환 및 대기열

한 번에 여러 파일을 선택하세요. 대기열에 추가되어 하나씩 처리됩니다. 개별적으로 다운로드하거나 나중에 일괄 패키지로 다운로드하세요.

언제든지 목록을 지우고 다시 시작하세요.

🎧 내장 미니 플레이어

다운로드하기 전에 변환된 오디오를 페이지에서 직접 미리 보세요. 시간을 절약하고 반복적인 다운로드를 방지합니다.

일시 정지, 탐색 – 빠른 미리보기.

자주 묻는 질문(FAQ)

어떤 오디오 형식이 지원되나요?

현재 MP3, WAV, FLAC, AAC(M4A), OGG(Opus), AIFF, WMA, WebM(Opus) 간 변환을 지원합니다. 비디오 파일(MP4, MKV, MOV, AVI, 3GP, TS)에서 오디오를 추출하고 이러한 형식으로 변환할 수도 있습니다. 시간이 지남에 따라 더 많은 형식이 추가될 예정입니다.

변환된 오디오 품질은 얼마나 좋나요?

품질은 전적으로 귀하의 설정에 따라 달라집니다.
- WAV 또는 AIFF의 경우: 원본 오디오 데이터와 동일한 무손실 PCM.
- MP3의 경우: 슬라이더를 사용하여 비트 전송률(64kbps ~ 320kbps)을 설정합니다. 높을수록 품질이 좋고 파일 크기가 커집니다.
- 샘플 속도 옵션을 사용하면 다운샘플링하여 파일 크기를 줄일 수 있습니다(예: 48kHz → 44.1kHz 또는 22kHz).
최상의 결과를 얻으려면 원래 샘플 속도와 스테레오를 유지하는 것이 좋습니다.

파일이 정말 업로드되지 않나요?

예, 100% 로컬 처리입니다. 우리는 Web Audio API를 사용하여 디코딩하고, lamejs를 사용하여 MP3를 인코딩하고, MediaRecorder for Opus를 사용합니다. 모든 작업은 브라우저의 메모리에서 이루어지며 네트워크 전송은 없습니다. 이 도구는 오프라인에서도 사용할 수 있습니다(페이지가 한 번 로드된 후).

일부 동영상 파일을 변환할 수 없는 이유는 무엇인가요?

브라우저는 지원하는 비디오 코덱만 디코딩할 수 있습니다. 예를 들어 일부 MKV 파일에는 HEVC(H.265) 비디오가 포함될 수 있으며 Safari 또는 Chrome은 해당 오디오 트랙을 지원하지 않을 수 있습니다. 보다 일반적인 코덱(예: H.264 + AAC)을 사용해 보세요. 문제가 지속되면 먼저 비디오 인코딩을 변환한 다음 오디오를 추출하십시오.

여러 파일을 한 번에 변환할 수 있나요?

네. 여러 파일을 선택하면 대기열에 나타납니다. "변환 시작"을 클릭하면 도구가 하나씩 처리합니다. 변환 중에 파일을 추가하거나 제거할 수 있지만 현재 처리 중인 파일은 수정할 수 없습니다.

대용량 파일을 변환하면 페이지가 멈추나요?

오디오 인코딩은 특히 대용량 파일이나 높은 비트 전송률의 경우 CPU 집약적입니다. 최대한 최적화하더라도 메인 스레드가 잠시 차단될 수 있습니다. 매우 큰 파일(예: 100MB 이상)의 경우 잠시 기다려 주십시오. 진행률 표시줄이 실시간으로 업데이트됩니다. 향후 버전에서는 보다 원활한 성능을 위해 Web Worker가 도입될 수 있습니다.
