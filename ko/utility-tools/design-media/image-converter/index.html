기술 사양

사용된 형식 및 기술:

  • SVG: 벡터 그래픽, 품질 손실 없이 확장 가능
  • PNG: 무손실, 투명도, 웹 그래픽에 이상적
  • JPEG: 손실 압축, 작은 파일 크기, 사진
  • WebP: 최신 형식, JPEG
    • 보다 압축률이 25-35% 향상되었습니다.
  • BMP: 비압축, 최대 품질
  • TIFF: 인쇄 산업 표준, 레이어 지원
  • HEIC/HEIF: Apple의 고효율 형식, JPEG
    • 보다 50% 더 작음
  • 크기 조정 및 변환: HTML5 Canvas API + 웹 작업자; DOMParser로 구문 분석된 SVG; heic2any를 사용한 HEIC; UTIF.js를 사용한 TIFF

참고: 모든 처리는 웹 작업자를 사용하여 인터페이스의 응답성을 유지합니다. 매우 큰 이미지의 경우 처리하는 데 몇 초가 걸릴 수 있지만 페이지가 정지되지는 않습니다.

자주 묻는 질문(FAQ)

로컬 이미지 리사이저를 사용하는 이유는 무엇입니까?

기존 온라인 도구에서는 서버에 이미지를 업로드해야 하므로 개인정보가 노출될 위험이 있습니다. 우리의 도구는 귀하의 브라우저에서 완전히 실행되며 귀하의 이미지는 귀하의 장치에 유지됩니다. 개인, 기밀 또는 민감한 이미지에 적합합니다.

이미지 크기를 조정할 수 있나요? 어떻게?

네! 크기 조정 모드(퍼센트, 고정 너비, 고정 높이 또는 자유 늘이기)를 선택하고 원하는 값을 입력한 다음 "처리"를 클릭합니다. 이 도구는 크기가 조정된 이미지를 로컬에서 생성합니다.

HEIC란 무엇이며 왜 변환하나요?

HEIC는 iPhone/iPad의 기본 사진 형식입니다. JPEG보다 더 나은 압축을 제공하지만 보편적으로 지원되지는 않습니다. 우리의 도구는 HEIC를 JPG/PNG로 변환하므로 어디서나 보고 사용할 수 있습니다.

SVG 변환을 지원하나요?

그렇죠. SVG 파일을 업로드하고 출력 래스터 형식(PNG, JPEG, WebP, BMP, TIFF)을 선택하세요. 필요한 크기를 정확히 얻도록 출력 크기를 설정할 수도 있습니다.

무료인가요?

네, 완전 무료입니다. 등록이나 숨겨진 수수료가 없습니다. 비공개로 이미지 크기를 조정하고 변환하세요.

어떤 브라우저가 지원되나요?

모든 최신 브라우저: Chrome 65+, Firefox 61+, Safari 11.1+, Edge 79+. 모바일 브라우저(iOS Safari, Android Chrome)도 작동합니다.
