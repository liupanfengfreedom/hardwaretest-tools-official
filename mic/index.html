在线麦克风测试工具使用教程

本工具支持实时检测麦克风音量、波形图、频率分析、增益调节等……

1. 打开在线麦克风检测页面
点击浏览器上方的“允许访问麦克风”，否则无法检测。

2. 选择你的麦克风设备
如果你有多个麦克风（USB 麦、耳机麦、主板麦克风等），在下拉菜单中选择正确的输入设备。

3. 检查麦克风音量
说话时观察实时音量条是否正常波动。如果音量过低或没有反应，可尝试：
- 调大系统输入音量
- 更换麦克风接口
- 检查是否被其他软件独占（如 Zoom、OBS、Discord）

4. 观看实时波形图
波形图可以帮助你判断声音是否顺畅：
- 声音正常 → 曲线平滑、有变化
- 噪音严重 → 波形密集抖动
- 麦克风损坏 → 波形固定不动

5. 查看频率分析
频谱图可用于判断音质：
- 高频消失 → 声音闷、模糊
- 低频过强 → 嗡嗡声明显
- 有尖刺峰值 → 可能存在电流声或底噪

6. 麦克风增益与降噪建议
若你声音太小，可适度增加麦克风增益，但需避免过高导致失真。
如有噪音，可启用：
- Windows 麦克风增强
- 噪声抑制
- 自动增益控制

7. 测试完成后的建议
- 若你的麦克风音质不佳，建议更换更高质量的外置麦克风。
- 如果你需要录音、直播、会议，请确保音量稳定在绿色-黄色之间。

常见麦克风问题排查

如音量小、杂音、权限问题、虚拟声卡冲突等。

1. 麦克风声音小或对方听不见
可能原因：
- 系统音量过低
- Windows 隐私权限未开启（设置 → 隐私 → 麦克风）
- 设备默认输入错误
解决方法：
- 在输入设备中选择正确的麦克风
- 将系统增益调至 80% 以上
- 关闭占用麦克风的软件

2. 出现杂音、电流声、嘶嘶声
原因可能包括：
- 3.5mm 接口接触不良
- 主板前置音频质量差
- USB 麦克风供电不稳
解决办法：
- 尝试连接主板后置接口
- 更换 USB 端口（避免 USB Hub）
- 使用带有独立声卡的麦克风

3. 麦克风有爆音、尖锐声
通常是输入增益过高或拾音器损坏。
解决方法：
- 调低麦克风增益
- 保持嘴与麦克风距离 10–15 cm
- 使用防喷罩减少爆破音

4. 麦克风延迟高
原因可能是：
- 浏览器采样率不同步
- 声卡驱动落后
- 在线会议软件后台占用
解决步骤：
- 将系统采样率统一为 44100 Hz
- 更新声卡驱动
- 关闭多余的录音软件

5. 虚拟声卡或变声器导致异常
如使用 iVCam、Voicemod、OBS 虚拟设备，可能造成无声或延迟。
解决方法：
- 在测试工具选择真实物理麦克风
- 临时卸载虚拟声卡进行排查

6. 麦克风完全无法使用
检查以下步骤：
- 设备管理器是否识别麦克风
- 是否被禁用（右键 → 启用）
- USB 数据线损坏或接触不良
- 在其他设备上测试是否可用
