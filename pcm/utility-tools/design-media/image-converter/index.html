Technical details

Formats and tech wey dis page dey use:

  • SVG: vector graphics, scalable without quality loss
  • PNG: lossless, transparency, ideal for web graphics
  • JPEG: lossy compression, small file size, photos
  • WebP: modern format, 25‑35% better compression than JPEG
  • BMP: uncompressed, maximum quality
  • TIFF: print industry standard, fit hold layers
  • HEIC/HEIF: Apple's high‑efficiency format, 50% smaller than JPEG
  • Resizing & conversion: HTML5 Canvas API + Web Workers; SVG parsed with DOMParser; HEIC with heic2any; TIFF with UTIF.js

Note:Dis page dey use Web Workers so the page no go hang. Very big image fit take small time, but e no go freeze the page.
