Características del Apple Magic Keyboard y del Apple Probador de teclado

Visualización completa del teclado · Compatible con Magic Keyboard

Distribución completa del teclado ISO de Apple. Ya sea que esté utilizando un Apple Magic Keyboard, un Magic Keyboard o un teclado integrado de AppleBook, cada pulsación de tecla resalta instantáneamente la tecla correspondiente. Admita teclas de función, modificadores y teclas internacionales para obtener comentarios intuitivos.

Detección precisa de vibración

Registre automáticamente la hora de cada pulsación de tecla. Cuando se presiona dos veces dentro de 80 ms, la entrada del registro se resalta en rojo para ayudar a identificar rápidamente posibles problemas de rebote del interruptor en su Magic Keyboard.

Consejo: Los intervalos de tecla normales suelen ser >100 ms. El parloteo rápido y anormal suele deberse a interruptores viejos.

Prueba de clave combinada y modificada

Pruebe individualmente Control, Opción, Comando, Fn y otros modificadores. Presione simultáneamente varias teclas para probar NKRO (se muestra el máximo de teclas simultáneas). Perfecto para evaluar los mecanismos de mariposa o tijera del Magic Keyboard.

Análisis de códigos clave y registros en tiempo real

Cada prensa y comunicado se registra con hora precisa, clave, código y valor, ideal para desarrolladores o usuarios avanzados que analizan los detalles de entrada o la latencia de Bluetooth en Magic Keyboard.

Compatible con todos los navegadores modernos

Construido con JavaScript básico, no se requieren complementos. Se ejecuta sin problemas en Chrome, Edge, Safari, Firefox. También funciona con Magic Keyboard conectado por Bluetooth.

Restablecer con un clic, repetir prueba

Haga clic en "Restablecer prueba" para borrar todos los recuentos y registros, lo que facilita la ejecución de varias rondas de prueba o la comparación de diferentes teclados (por ejemplo, Magic Keyboard frente a terceros).

Preguntas frecuentes (FAQ)

¿Cómo pruebo las teclas de mi Apple Magic Keyboard?

Presiona cualquier tecla de tu Magic Keyboard mientras observas la distribución del teclado en el centro de la página. La clave correspondiente se resaltará, el panel de estado mostrará el nombre de la clave, el código clave y el valor, y el área de registro registrará cada pulsación y liberación con marcas de tiempo. Si una tecla no responde, puede indicar un problema de hardware o de conexión Bluetooth.

¿Qué es la detección de chatter? ¿Cómo puedo saber si una tecla del Magic Keyboard vibra?

La detección de conversaciones monitorea el intervalo de tiempo entre dos pulsaciones de la misma tecla. Si el intervalo es inferior a 80 milisegundos (lo que normalmente indica un doble clic rápido y anormal), la entrada del registro se resalta en rojo y se etiqueta como una alerta de conversación. Esto le ayuda a identificar posibles problemas de rebote de interruptores, comunes en algunos mecanismos de teclado.

¿Por qué mi Apple Magic Keyboard no muestra respuesta en el registro cuando presiono una tecla?

Asegúrese de que el área de prueba esté activa (simplemente presione cualquier tecla directamente). Si aún no hay respuesta, podría ser un problema de permisos del navegador o un problema de hardware. Intente actualizar la página o utilizar otro navegador (se recomienda Chrome, Edge). También verifique si su sistema operativo reconoce el teclado, especialmente para conexiones Bluetooth. También se admiten teclados mágicos con cable.

¿Cómo puedo probar las teclas modificadas (Comando, Opción, etc.)?

Simplemente presione la tecla modificada (por ejemplo, Comando izquierdo, Opción derecha). La tecla correspondiente en el teclado en pantalla se iluminará y el panel de estado mostrará el nombre y el código de la tecla. Puede presionar varios modificadores simultáneamente para probar el reconocimiento de teclas combinadas.

¿Qué diseños de teclado Apple son compatibles? ¿Es compatible el teclado mágico de Apple?

Esta herramienta utiliza una distribución de teclado ISO estándar de Apple, que cubre todas las teclas principales, incluidas las teclas de función (F1-F12), teclas numéricas, letras, puntuación y los modificadores de la fila inferior (Control, Opción, Comando) y teclas de flecha. Es totalmente compatible con Apple Magic Keyboard (incluida la versión con teclado numérico), Magic Keyboard, teclados integrados AppleBook y otros teclados Apple de terceros.

¿Cuál es el significado del máximo de teclas simultáneas (NKRO) para Magic Keyboard?

El número máximo de teclas simultáneas indica cuántas teclas se pueden presionar simultáneamente y aún así reconocerse correctamente. Para jugadores o usuarios con ataques complejos, un valor NKRO alto es importante. Magic Keyboard normalmente admite la sustitución de 6 teclas. Esta herramienta registra la cantidad actual de teclas presionadas y muestra el máximo histórico durante la prueba, lo que le brinda información sobre la capacidad de concurrencia real de su teclado.
