Política de privacidad

Última actualización: marzo de 2026

1. Introducción

Este sitio web respeta su privacidad y se compromete a proteger su información. Esta Política de privacidad explica cómo recopilamos, utilizamos y salvaguardamos la información cuando visita nuestro sitio web.

2. Información que recopilamos

No requerimos registro de usuario. Sin embargo, cierta información puede recopilarse automáticamente:

  • dirección IP
  • Tipo y versión del navegador
  • Información del dispositivo
  • Páginas visitadas y tiempo invertido
  • Sitio web de referencia

3. Galletas

Este sitio web utiliza cookies para mejorar la experiencia del usuario y analizar el tráfico. Las cookies son pequeños archivos de texto almacenados en su dispositivo.

4. Google Analytics

Utilizamos Google Analytics para analizar el uso del sitio web y los patrones de tráfico. Google podrá recopilar información según su propia Política de privacidad.

5. Servicios de publicidad

Este sitio web puede mostrar anuncios a través de redes publicitarias de terceros como Google AdSense. Estos servicios pueden utilizar cookies y tecnologías similares para mostrar anuncios personalizados según sus intereses.

6. Cómo utilizamos la información

  • Para mejorar la funcionalidad del sitio web
  • Para analizar el tráfico y las tendencias de uso.
  • Para mantener la seguridad y prevenir abusos
  • Para publicar anuncios relevantes

7. Intercambio de datos

No vendemos información personal. Los datos pueden ser procesados ​​por proveedores externos de confianza estrictamente con fines de análisis, publicidad y alojamiento.

8. Retención de datos

La información recopilada se conserva solo el tiempo necesario para análisis, seguridad y cumplimiento legal.

9. Tus derechos

Dependiendo de su ubicación, es posible que tenga derechos según las leyes de privacidad aplicables, incluido el derecho a acceder, eliminar o restringir el procesamiento de su información personal.

10. Privacidad de los niños

Este sitio web no está dirigido a niños menores de 13 años.

11. Actualizaciones

Podemos actualizar esta Política de privacidad de vez en cuando. El uso continuado del sitio web constituye la aceptación de cualquier cambio.

12. Contactoo

Si tiene preguntas sobre esta Política de privacidad, por favor contacte con: contact@starryring.com
