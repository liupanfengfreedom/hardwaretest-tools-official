Häufig gestellte Fragen (FAQ)

Was ist Passwortentropie und warum ist sie wichtig?

Password-Entropie misst Zufälligkeit und Unvorhersehbarkeit, ausgedrückt in Bits. Höhere Entropie bedeutet stärkere Passwörter, die schwerer zu knacken sind. Formel: Entropie = log₂(charset_size ^ passwort_länge). Beispielsweise ergibt ein 12-stelliges Passwort aus einem Satz von 94 Zeichen etwa 78,5 Bit Entropie.

Warum sollte ich bestimmte Zeichen ausschließen?

Das Ausschließen von Zeichen verbessert die Benutzerfreundlichkeit: 1) Vermeiden Sie Verwirrung (0/O, 1/l, 5/S). 2) Einige Systeme schränken spezielle Symbole ein. 3) Einfachere Eingabe auf mobilen Geräten. Beachten Sie jedoch, dass das Ausschließen von Zeichen die Zeichensatzgröße verringert und die Sicherheit geringfügig beeinträchtigen kann.

Sind generierte Passwörter wirklich sicher?

Ja. CyberPass Pro verwendet die Crypto.getRandomValues() API des Browsers – einen kryptografisch sicheren Zufallszahlengenerator. Alle Vorgänge erfolgen lokal; Passwörter verlassen niemals Ihr Gerät. Bei ausreichend großem Zeichensatz sind generierte Passwörter sehr sicher.

Wie soll ich generierte Passwörter speichern?

Verwenden Sie immer einen seriösen Passwort-Manager (Bitwarden, 1Password, LastPass usw.), um Passwörter sicher zu speichern. Speichern Sie Passwörter niemals in Klartextdateien und geben Sie sie niemals auf unsichere Weise weiter. Dieses Tool bietet Kopier- und Exportfunktionen für eine einfache Integration mit Passwort-Managern.
