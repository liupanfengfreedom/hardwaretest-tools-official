Häufig gestellte Fragen (FAQ)

Warum ist mein Symbol bei bestimmten Größen verschwommen?

Unscharfe Symbole sind normalerweise auf eine unzureichende Auflösung des Quellbilds zurückzuführen. Wir empfehlen:

  • • Verwenden Sie ein Quellbild mit mindestens 1024×1024 Pixeln
  • • Bevorzugen Sie Vektor (SVG) oder hochauflösendes PNG
  • • Stellen Sie sicher, dass die Schlüsselelemente Ihres Symbols groß und deutlich sind
  • • Vermeiden Sie übermäßige Details im Quellbild – bei kleinen Größen gehen Details verloren

Wie kann ich eine konsistente Anzeige auf verschiedenen Plattformen sicherstellen?

Plattformunterschiede sind normal, aber Sie können mit diesen Tipps optimieren:

  • • Bewahren Sie wichtige Inhalte im zentralen Bereich aufsichere Zone(Kanten vermeiden)
  • • Verwenden Sie kontrastreiche Designs für eine klare Erkennung bei kleinen Größen
  • • Überprüfen Sie die Echtzeitvorschau für beide Plattformen im rechten Bereich
  • • iOS fügt automatisch abgerundete Ecken hinzu; Android-Formen variieren je nach Thema – entwerfen Sie entsprechend

Unterstützt es transparente Hintergründe?

Ja, transparente Hintergründe werden vollständig unterstützt:

  • • Transparente Bereiche in hochgeladenen PNGs bleiben erhalten
  • • Erstellte Symbole behalten die ursprüngliche Transparenz bei
  • • Für Plattformen, die undurchsichtige Hintergründe erfordern (z. B. iOS App Store), wird automatisch ein weißer Hintergrund hinzugefügt
  • • Überlegen Sie, wie Transparenz auf verschiedenen Hintergründen aussehen wird

Welche Dateiformate kann ich generieren?

Derzeit werden Symbole hauptsächlich im PNG-Format generiert:

  • • Alle Symbole werden als PNG generiert (empfohlenes plattformübergreifendes Format)
  • • PNG unterstützt transparente Hintergründe und hochwertige Komprimierung
  • • Für Android bietet WebP möglicherweise eine bessere Komprimierung, erfordert jedoch eine separate Konvertierung
  • • iOS App Store-Symbole erfordern PNG ohne Transparenz

Ist dieses Tool kostenlos? Gibt es Einschränkungen?

Ja, App Icon Studio ist völlig kostenlos:

  • • Kostenlose Nutzung, keine Registrierung oder Anmeldung erforderlich
  • • Keine Nutzungsbeschränkungen
  • • Keine Bildgrößenbeschränkung (die Verarbeitung großer Bilder kann jedoch länger dauern)
  • • Die gesamte Verarbeitung erfolgt im Browser, es fallen keine Serverkosten an
  • • Zukünftige erweiterte Funktionen können hinzugefügt werden, die Kernfunktionalität bleibt jedoch kostenlos

Wie integriere ich generierte Icons in mein Projekt?

Integrationsschritte variieren je nach Plattform:

  • iOS (Xcode):Ziehen Sie Symbole in den AppIcon-Satz in Assets.xcassets
  • Android (Android Studio):Symbole in res/mipmap-*dpi/-Ordner kopieren
  • React Native:Verwenden Sie das React-Native-Make-Tool oder platzieren Sie es manuell in den Verzeichnissen ios/ und android/
  • Flutter:Symbolkonfiguration in pubspec.yaml aktualisieren und in den entsprechenden Verzeichnissen ablegen

Die generierte ZIP-Datei ist zur einfachen Verwendung bereits nach Plattform und Größe geordnet.
