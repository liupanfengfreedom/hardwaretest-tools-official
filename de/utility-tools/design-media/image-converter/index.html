Technische Spezifikationen

Verwendete Formate und Technologien:

  • SVG: Vektorgrafiken, skalierbar ohne Qualitätsverlust
  • PNG: verlustfrei, Transparenz, ideal für Webgrafiken
  • JPEG: verlustbehaftete Komprimierung, kleine Dateigröße, Fotos
  • WebP: modernes Format, 25–35 % bessere Komprimierung als JPEG
  • BMP: unkomprimiert, maximale Qualität
  • TIFF: Druckindustriestandard, unterstützt Ebenen
  • HEIC/HEIF: Das hocheffiziente Format von Apple, 50 % kleiner als JPEG
  • Größenänderung und Konvertierung: HTML5 Canvas API + Web Workers; SVG mit DOMParser analysiert; HEIC mit heic2any; TIFF mit UTIF.js

Hinweis:Bei der gesamten Verarbeitung werden Web Worker verwendet, um die Reaktionsfähigkeit der Schnittstelle aufrechtzuerhalten. Bei sehr großen Bildern kann die Verarbeitung einige Sekunden dauern, die Seite wird jedoch nicht eingefroren.

Häufig gestellte Fragen (FAQ)

Warum einen lokalen Bild-Resizer verwenden?

Herkömmliche Online-Tools erfordern das Hochladen von Bildern auf einen Server, was die Privatsphäre gefährdet. Unser Tool läuft vollständig in Ihrem Browser – Ihre Bilder bleiben auf Ihrem Gerät. Perfekt für persönliche, vertrauliche oder sensible Bilder.

Kann ich die Größe von Bildern ändern? Wie?

Ja! Wählen Sie einen Größenänderungsmodus (Prozent, feste Breite, feste Höhe oder freie Dehnung), geben Sie die gewünschten Werte ein und klicken Sie auf „Verarbeiten“. Das Tool generiert lokal verkleinerte Bilder.

Was ist HEIC und warum wird es konvertiert?

HEIC ist das Standardfotoformat auf iPhones/iPads. Es bietet eine bessere Komprimierung als JPEG, wird jedoch nicht allgemein unterstützt. Unser Tool konvertiert HEIC in JPG/PNG, sodass Sie sie überall ansehen und verwenden können.

Unterstützt es die SVG-Konvertierung?

Absolut. Laden Sie SVG-Dateien hoch und wählen Sie ein Ausgabe-Rasterformat (PNG, JPEG, WebP, BMP, TIFF). Sie können auch die Ausgabeabmessungen festlegen, um genau die Größe zu erhalten, die Sie benötigen.

Ist es kostenlos?

Ja, völlig kostenlos. Keine Registrierung, keine versteckten Gebühren. Ändern Sie einfach die Größe Ihrer Bilder und konvertieren Sie sie privat.

Welche Browser werden unterstützt?

Alle modernen Browser: Chrome 65+, Firefox 61+, Safari 11.1+, Edge 79+. Mobile Browser (iOS Safari, Android Chrome) funktionieren ebenfalls.
