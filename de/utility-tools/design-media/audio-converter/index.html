Häufig gestellte Fragen (FAQ)

Welche Audioformate werden unterstützt?

Derzeit unterstützen wir die Konvertierung zwischen: MP3, WAV, FLAC, AAC (M4A), OGG (Opus), AIFF, WMA, WebM (Opus). Sie können auch Audio aus Videodateien (MP4, MKV, MOV, AVI, 3GP, TS) extrahieren und in eines dieser Formate konvertieren. Weitere Formate werden im Laufe der Zeit hinzugefügt.

Wie gut ist die konvertierte Audioqualität?

Qualität hängt vollständig von Ihren Einstellungen ab:
- Für WAV oder AIFF: verlustfreies PCM, identisch mit den Original-Audiodaten.
- Für MP3: Verwenden Sie den Schieberegler, um die Bitrate einzustellen (64 kbps bis 320 kbps) – höher ergibt eine bessere Qualität und größere Dateien.
– Mit den Optionen für die Abtastrate können Sie ein Downsampling durchführen, um die Dateigröße zu reduzieren (z. B. 48 kHz → 44,1 kHz oder 22 kHz).
Wir empfehlen, die ursprüngliche Abtastrate und Stereo beizubehalten, um optimale Ergebnisse zu erzielen.

Werden Dateien wirklich nicht hochgeladen?

Ja, 100 % lokale Verarbeitung. Wir verwenden die Web Audio API zum Dekodieren, lamejs zum Kodieren von MP3 und MediaRecorder für Opus. Alles geschieht im Speicher Ihres Browsers – keine Netzwerkübertragung. Sie können dieses Tool sogar offline verwenden (nachdem die Seite einmal geladen wurde).

Warum kann ich einige Videodateien nicht konvertieren?

Browser können nur Videocodecs dekodieren, die sie unterstützen. Beispielsweise können einige MKV-Dateien HEVC-Videos (H.265) enthalten, und Safari oder Chrome unterstützen die Audiospur möglicherweise nicht. Versuchen Sie es mit einem gängigeren Codec (wie H.264 + AAC). Wenn die Probleme weiterhin bestehen, konvertieren Sie zuerst die Videokodierung und extrahieren Sie dann den Ton.

Kann ich mehrere Dateien gleichzeitig konvertieren?

Ja. Wählen Sie mehrere Dateien aus – sie erscheinen in der Warteschlange. Klicken Sie auf „Konvertierung starten“ und das Tool verarbeitet sie nacheinander. Sie können während der Konvertierung Dateien hinzufügen oder entfernen, Dateien, die gerade verarbeitet werden, können jedoch nicht geändert werden.

Wird die Seite beim Konvertieren großer Dateien einfrieren?

Die Audiokodierung ist CPU-intensiv, insbesondere bei großen Dateien oder hohen Bitraten. Obwohl wir so viel wie möglich optimieren, kann es vorkommen, dass der Hauptthread kurzzeitig blockiert. Bei sehr großen Dateien (z. B. >100 MB) haben Sie bitte etwas Geduld – der Fortschrittsbalken wird in Echtzeit aktualisiert. In einer zukünftigen Version werden möglicherweise Web Worker für eine reibungslosere Leistung eingeführt.
