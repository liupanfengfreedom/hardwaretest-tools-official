Apple Magic Keyboard und Apple Tastaturtester-Funktionen

Vollständige Tastaturvisualisierung · Kompatibel mit Magic Keyboard

Komplettes Apple ISO-Tastaturlayout. Unabhängig davon, ob Sie ein Apple Magic Keyboard, ein Magic Keyboard oder eine integrierte AppleBook-Tastatur verwenden, wird bei jedem Tastendruck sofort die entsprechende Taste hervorgehoben. Unterstützt Funktionstasten, Modifikatoren und internationale Tasten für intuitives Feedback.

Präzise Vibrationserkennung

Versieht jeden Tastendruck automatisch mit einem Zeitstempel. Wenn innerhalb von 80 ms zwei Tastendrücke erfolgen, wird der Protokolleintrag rot hervorgehoben, damit Sie potenzielle Probleme mit dem Schalterspringen auf Ihrem Magic Keyboard schnell erkennen können.

Tipp: Normale Tastenintervalle betragen normalerweise >100 ms. Ungewöhnlich schnelles Rattern wird häufig durch alternde Schalter verursacht.

Modifikator- und Kombinationstastentest

Testen Sie Control, Option, Command, Fn und andere Modifikatoren einzeln. Drücken Sie gleichzeitig mehrere Tasten, um NKRO zu testen (maximal gleichzeitige Tasten werden angezeigt). Perfekt zum Testen der Butterfly- oder Scherenmechanismen des Magic Keyboard.

Echtzeitprotokoll- und Schlüsselcodeanalyse

Jedes Drücken und Loslassen wird mit präziser Uhrzeit, Taste, Code und Wert protokolliert – ideal für Entwickler oder fortgeschrittene Benutzer, die Eingabedetails oder Bluetooth-Latenz auf dem Magic Keyboard analysieren.

Kompatibel mit allen modernen Browsern

Erstellt mit Vanilla-JavaScript, keine Plugins erforderlich. Läuft reibungslos auf Chrome, Edge, Safari, Firefox. Funktioniert auch mit dem über Bluetooth verbundenen Magic Keyboard.

Ein-Klick-Zurücksetzen, Test wiederholen

Klicken Sie auf „Test zurücksetzen“, um alle Zählungen und Protokolle zu löschen. So können Sie problemlos mehrere Testrunden durchführen oder verschiedene Tastaturen vergleichen (z. B. Magic Keyboard vs. Tastaturen von Drittanbietern).

Häufig gestellte Fragen (FAQ)

Wie teste ich die Tasten auf meinem Apple Magic Keyboard?

Drücken Sie eine beliebige Taste auf Ihrem Magic Keyboard, während Sie sich das Tastaturlayout in der Mitte der Seite ansehen. Die entsprechende Taste wird hervorgehoben, im Statusfeld werden der Tastenname, der Schlüsselcode und der Wert angezeigt, und im Protokollbereich wird jedes Drücken und Loslassen mit Zeitstempeln aufgezeichnet. Wenn eine Taste nicht reagiert, kann dies auf ein Hardwareproblem oder ein Problem mit der Bluetooth-Verbindung hinweisen.

Was ist Flattererkennung? Wie kann ich feststellen, ob eine Taste der Magic-Tastatur klappert?

Die Chatter-Erkennung überwacht das Zeitintervall zwischen zwei Drücken derselben Taste. Wenn das Intervall weniger als 80 Millisekunden beträgt (was typischerweise auf ungewöhnlich schnelles Doppelklicken hinweist), wird der Protokolleintrag rot hervorgehoben und als Chatter-Warnung gekennzeichnet. Dies hilft Ihnen, potenzielle Probleme mit der Schalterprellung zu erkennen, die bei einigen Tastaturmechanismen häufig auftreten.

Warum zeigt mein Apple Magic Keyboard keine Reaktion im Protokoll, wenn ich eine Taste drücke?

Stellen Sie sicher, dass der Testbereich aktiv ist (drücken Sie einfach direkt eine beliebige Taste). Wenn immer noch keine Antwort erfolgt, liegt möglicherweise ein Browser-Berechtigungsproblem oder ein Hardwareproblem vor. Versuchen Sie, die Seite zu aktualisieren oder einen anderen Browser zu verwenden (Chrome, Edge empfohlen). Überprüfen Sie außerdem, ob Ihr Betriebssystem die Tastatur erkennt, insbesondere bei Bluetooth-Verbindungen. Kabelgebundene Magic-Tastaturen werden ebenfalls unterstützt.

Wie kann ich Zusatztasten (Befehl, Option usw.) testen?

Drücken Sie einfach die Zusatztaste (z. B. linke Befehlstaste, rechte Wahltaste). Die entsprechende Taste auf der Bildschirmtastatur leuchtet auf und im Statusfeld werden der Tastenname und der Code angezeigt. Sie können mehrere Modifikatoren gleichzeitig drücken, um die Erkennung von Kombinationstasten zu testen.

Welche Apple-Tastaturlayouts werden unterstützt? Ist das Apple Magic Keyboard kompatibel?

Dieses Tool verwendet ein standardmäßiges ISO-Apple-Tastaturlayout und deckt alle wichtigen Tasten ab, einschließlich Funktionstasten (F1-F12), Zifferntasten, Buchstaben, Satzzeichen sowie Modifikatoren in der unteren Reihe (Strg, Option, Befehl) und Pfeiltasten. Es ist vollständig kompatibel mit dem Apple Magic Keyboard (einschließlich der Version mit Ziffernblock), dem Magic Keyboard, den integrierten AppleBook-Tastaturen und anderen Apple-Tastaturen von Drittanbietern.

Welche Bedeutung hat die maximale Anzahl gleichzeitiger Tasten (NKRO) für Magic Keyboard?

Maximal gleichzeitige Tasten geben an, wie viele Tasten gleichzeitig gedrückt werden können und trotzdem korrekt erkannt werden. Für Gamer oder Benutzer mit komplexen Verknüpfungen ist ein hoher NKRO-Wert wichtig. Magic Keyboard unterstützt normalerweise 6-Tasten-Rollover. Dieses Tool zeichnet die aktuelle Anzahl der gedrückten Tasten auf und zeigt das historische Maximum während des Tests an, sodass Sie einen Einblick in die tatsächliche Parallelitätsfähigkeit Ihrer Tastatur erhalten.
